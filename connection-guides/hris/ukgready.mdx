If you've been directed to StackOne to integrate with UKG Ready, the following steps will help you understand the process and any necessary actions to configure successful integration.

This guidance assumes you have Admin privileges for your UKG Ready account.

## Getting Company Short Name and API Key

<Steps>
  <Step title="Login to UKG Ready dashboard">
    Login to your UKG Ready dashboard and click ≡ at the top-left corner.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="UKG Ready Dashboard" src="/images/ukgready/image1.png" />
    </Frame>
  </Step>

  <Step title="Navigate to Company Setup">
    Expand ⚙️Settings menu and select Global Setup > Company Setup option.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Company Setup" src="/images/ukgready/image2.png" />
    </Frame>
  </Step>

  <Step title="Copy Company Short Name">
    Under the Company Setup, copy your Company Short Name. If it is not available in that section, please enter the Company Short Name and click `Save` button.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Company Short Name" src="/images/ukgready/image3.png" />
    </Frame>
  </Step>

  <Step title="Generate API Key">
    In the Login Info tab, locate the API Keys section. If you don't already have an API key, click the `Generate` button. Copy the generated API key to a safe place.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Generate API Key" src="/images/ukgready/image4.png" />
    </Frame>
  </Step>
</Steps>

## Utilizing API Key and Company Short Name

<Steps>
  <Step title="Link Account">
    Upon reaching the Link Account page, enter the API key, your UKG Ready username/password, and your Company Short Name. Proceed by clicking the `Connect` button to establish the connection.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Link Account" src="/images/ukgready/image5.png" />
    </Frame>
  </Step>
</Steps>
