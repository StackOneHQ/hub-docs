If you've been directed to StackOne to integrate with Sage HR, the following steps will help you understand the process and any necessary actions to configure successful integration.

This guidance assumes you have Admin privileges for your Sage HR account.

## Getting Company Subdomain and generating API Key

<Steps>
  <Step title="Login to Sage HR">
    Login to your Sage HR account and copy the subdomain from the URL bar in your web browser.
  </Step>
  <Step title="Navigate to Settings">
    To retrieve your Sage HR API key, click on your username in the top-right corner and select Settings.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Sage HR Settings" src="/images/sage/image1.png" />
    </Frame>
  </Step>

  <Step title="Enable API Access">
    Click Integrations and select API. Then click ENABLE API ACCESS to activate access to API.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Enable API Access" src="/images/sage/image2.png" />
    </Frame>
  </Step>

  <Step title="Copy API Key">
    Copy your generated API key to a safe place.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Copy API Key" src="/images/sage/image3.png" />
    </Frame>
  </Step>
</Steps>

## Linking your Sage Account

<Steps>
  <Step title="Enter API Key and Subdomain">
    Upon reaching the Link Account page, enter the gathered details and proceed by clicking the `Connect` button.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Link Account" src="/images/sage/image4.png" />
    </Frame>
  </Step>
</Steps>
