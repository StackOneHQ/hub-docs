</Warning>

If you've been directed to StackOne to integrate with Breathe HR, the following steps will help you understand the process and any necessary actions to configure successful integration.

## Getting your API Key

<Steps>
  <Step title="Log in to Breathe HR">
    Log in to your Breathe HR dashboard and click the Configure tab > Settings.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Breathe HR Dashboard" src="/images/breathehr/image1.png" />
    </Frame>
  </Step>

  <Step title="Locate API Setup">
    Inside the new page, locate the Integrations section and click API Setup.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="API Setup" src="/images/breathehr/image2.png" />
    </Frame>
  </Step>

  <Step title="Enable and Copy API Keys">
    Enable the API keys; now copy the API key for production & your sandbox to a safe place.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="API Keys" src="/images/breathehr/image3.png" />
    </Frame>
  </Step>
</Steps>

## Connecting with StackOne

<Steps>
  <Step title="Enter API Key">
    Enter one of your API keys (depending on the environment you want to connect) and proceed by clicking the `Connect` button.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Connect Button" src="/images/breathehr/image4.png" />
    </Frame>
  </Step>
</Steps>
