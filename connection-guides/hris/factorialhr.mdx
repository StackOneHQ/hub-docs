If you've been directed to StackOne to integrate with Factorial HR, the following steps will help you understand the process and any necessary actions to configure a successful integration.


## Create your Factorial HR API Key

<Steps>
  <Step title="Navigate to API Keys">
      In FactorialHR, click **Settings** in the side menu, scroll down to **Advanced configuration** then select **API Keys**.
  
      <Frame>
          <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Navigate to API Keys" src="/images/factorialhr/image4.png" />
      </Frame>
  </Step>
  <Step title="Create a new API Key">
      Click **New API Key** to create a new API Key.
  
      <Frame>
          <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Create a new API Key" src="/images/factorialhr/image5.png" />
      </Frame>
  </Step>
  <Step title="Name the API Key">
      Name the API Key then click **Get my API key**.
  
      <Frame>
          <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Name the API Key" src="/images/factorialhr/image6.png" />
      </Frame>
  </Step>
  <Step title="Copy your API Key">
      Copy your API Key and store it safely to be used in a later step.
  
      <Frame>
          <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Copy your API Key" src="/images/factorialhr/image7.png" />
      </Frame>
  </Step>
</Steps>

## Connecting with StackOne

Upon reaching the Link Account page, enter the **API Key** from the previous steps:

Proceed by clicking the Connect button.

<Frame>
    <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Enter Credentials" src="/images/factorialhr/image8.png" />
</Frame>


## Linking your Account

Enter the required fields from the previous steps in your connection form.
