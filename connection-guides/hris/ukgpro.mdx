If you've been directed to StackOne to integrate with UKG Pro, the following steps will help you understand the process and any necessary actions to configure successful integration.

This guidance assumes you have Admin privileges for your UKG Pro account.

## Getting your UKG Company Domain

<Steps>
  <Step title="Login to UKG Pro">
    Login to your UKG Pro account and search for Web Services in the search bar at the top right.

    <Frame>
      <img
        className="rounded-md"
        style={{ margin:"0 auto",border:"1px solid #efefef" }}
        alt="Web Services Search"
        src="/images/ukgpro/image1.png"
      />
    </Frame>
  </Step>
  <Step title="Copy Subdomain">
    Copy the subdomain from any Web Services URL. For example, if your URL is `http://Servicet.ultipro.com/services/BIDataService`, copy `Servicet`.

    <Frame>
      <img
        className="rounded-md"
        style={{ margin:"0 auto",border:"1px solid #efefef" }}
        alt="Subdomain Example"
        src="/images/ukgpro/image2.png"
      />
    </Frame>
  </Step>
</Steps>

<Steps>
  <Step title="Search for Service Account Administration">

## Creating a Service Account

<Steps>
  <Step title="Add Service Account">
    On the UKG Pro Service Account Administration page, click âž• add in the upper right corner.

    <Frame>
      <img
        className="rounded-md"
        style={{ margin:"0 auto",border:"1px solid #efefef" }}
        alt="Add Service Account"
        src="/images/ukgpro/image5.png"
      />
    </Frame>
  </Step>
  <Step title="Enter Admin Details">
    Enter your admin's username and email address.

    <Frame>
      <img
        className="rounded-md"
        style={{ margin:"0 auto",border:"1px solid #efefef" }}
        alt="Admin Details"
        src="/images/ukgpro/image6.png"
      />
    </Frame>
  </Step>
  <Step title="Assign Permissions">
    Scroll down to the Web Service table below and check the View permission for:

    - Personnel Integration
    - Employee Job History Details
    - Employee Employment Information
    - Employee Person Details
    - Employee Compensation Details
    - Company Configuration Integration

    <Frame>
      <img
        className="rounded-md"
        style={{ margin:"0 auto",border:"1px solid #efefef" }}
        alt="Assign Permissions"
        src="/images/ukgpro/image7.png"
      />
    </Frame>
  </Step>
  <Step title="Save and Generate Password">
    Click Save to generate a new password. Save this password in a secure location.

    <Frame>
      <img
        className="rounded-md"
        style={{ margin:"0 auto",border:"1px solid #efefef" }}
        alt="Generate Password"
        src="/images/ukgpro/image8.png"
      />
    </Frame>
  </Step>
  <Step title="Save Service User Credentials">
    Make a note of the service account credentials:

    - Service Account Username
    - Service Account Password
    - Service Account User API Key (shown below on the _Service Account Administration_ page)

    <Frame>
      <img
        className="rounded-md"
        style={{ margin:"0 auto",border:"1px solid #efefef" }}
        alt="Service User API Key"
        src="/images/ukg-pro-service-user-api-key.png"
      />
    </Frame>
  </Step>
</Steps>

## Connecting to StackOne

<Steps>
  <Step title="Enter Details">
    Enter your service account username and password from Step 3. Enter your API key from Step 2. Enter your web service domain from Step 1.

      />
    </Frame>
  </Step>
</Steps>
