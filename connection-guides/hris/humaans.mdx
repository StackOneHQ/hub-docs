If you've been directed to StackOne to integrate with Humaans, the following steps will help you understand the process and any necessary actions to configure successful integration.

This guidance assumes you have API access permissions for your Humaans account:

- click Settings > Permissions
- choose your role group, e.g., Owners
- verify the Workspace permissions section has API access enabled

## Step 1: Generating API Key

<Steps>
  <Step title="Open API Access Tokens">
    Click the expandable menu â‹® at the top left corner of your screen, beside your name. Then click [API access tokens](https://app.humaans.io).
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="API Access Tokens" src="/images/humaans/image1.png" />
    </Frame>
  </Step>

  <Step title="Generate New Token">
    Create a new API token by clicking the Generate new token button.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Generate New Token" src="/images/humaans/image2.png" />
    </Frame>
  </Step>

  <Step title="Assign a Label and Enable Scopes">
    Assign a label to your token and enable all of the scopes. Click Create access token.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Assign Label and Enable Scopes" src="/images/humaans/image3.png" />
    </Frame>
  </Step>

  <Step title="Copy the API Token">
    Copy the API token to a safe place.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Copy API Token" src="/images/humaans/image4.png" />
    </Frame>
  </Step>
</Steps>

## Step 2: Use the API Key in the StackOne Hub

<Steps>
  <Step title="Enter API Key and Connect">
    Proceed to the Account page to establish a connection between Humaans and StackOne. Input your API key and proceed by clicking the `Connect` button.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Link Account" src="/images/humaans/image5.png" />
    </Frame>
  </Step>
</Steps>
