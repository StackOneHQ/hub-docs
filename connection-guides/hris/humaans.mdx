<Note>
  Administrator access is required in your Humaans account to generate API tokens.
</Note>

<Steps>
  <Step title="Verify API Access Permissions">
    Before creating a token, ensure you have the correct permissions:
    - Click **Settings** → **Permissions**
    - Select your role group (e.g., Owners)
    - Verify that **API access** is enabled in the Workspace permissions section

    <Frame>
      <img src="/images/humaans/api_access_permissions.gif" style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="API Access Permissions" />
    </Frame>
  </Step>

  <Step title="Access API Tokens Page">
    Click the expandable menu ⋮ at the top left corner, then select **API access tokens**.

    <Frame>
      <img src="/images/humaans/image1.png" style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="API Access Tokens" />
    </Frame>
  </Step>

  <Step title="Create New Token">
    Click **Generate new token**.

    <Frame>
      <img src="/images/humaans/image2.png" style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Generate New Token" />
    </Frame>
  </Step>

  <Step title="Configure Token">
    Assign a descriptive label to your token and enable all required scopes. Click **Create access token**.

    <Frame>
      <img src="/images/humaans/image3.png" style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Configure Token" />
    </Frame>
  </Step>

  <Step title="Save API Token">
    Copy the API token immediately and store it securely — it won't be shown again.

    <Frame>
      <img src="/images/humaans/image4.png" style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Copy API Token" />
    </Frame>
  </Step>
</Steps>

## Linking your Humaans Account

Enter this field in your connection form:
- **API Key** (from Step 5)
