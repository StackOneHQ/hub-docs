If you've been directed to StackOne to integrate with Notion, the following steps will help you understand the process and any necessary actions to configure a successful integration.

## Log in to Notion

Log in to your Notion account.

    <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Log in to Notion" src="/images/notion/image2.png" />

## Create your Notion API Key

<Steps>
  <Step title="Navigate to Workspace Settings">
      Select your workspace and than click on `Settings` button.
  
      <Frame>
          <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Navigate to Workspace Settings" src="/images/notion/image3.png" />
      </Frame>
  </Step>
  <Step title="Navigate to Manage Integration">
      In the pop-up select `Account > Connections` and Click on `Develop or manage integrations` button to open [Manage Integration Page](https://www.notion.so/profile/integrations)
  
      <Frame>
          <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Navigate to Manage Integration" src="/images/notion/image4.png" />
      </Frame>
  </Step>
  <Step title="Create a new API Key">
      Click on `New integration` button under `Integrations` menu to create new API Key Integration.
  
      <Frame>
          <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Create a new API Key" src="/images/notion/image5.png" />
      </Frame>
  </Step>
  <Step title="Enter API Key Details">
      Enter the API Key name, select the workspace, select <b>Type</b> as `Internal` from dropdown, and click on `Save` button.
  
      <Frame>
          <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Enter API Key Details" src="/images/notion/image6.png" />
      </Frame>
  </Step>
  <Step title="Navigate to Configure API Integration Settings">
      Click on `Configure integration settings` button.
  
      <Frame>
          <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Navigate to Configure API Key Integration Settings" src="/images/notion/image7.png" />
      </Frame>
  </Step>
  <Step title="Configure API Integration Settings">
      Make sure you have check marked the following permissions under `Configuration > Capabilities > Content Capabilities` section:-
        <ol type="a">
        <li>Read content</li>
        <li>Update content</li>
        <li>Insert content</li>
       </ol>
  
      <Frame>
          <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Check mark required permissions" src="/images/notion/image8.png" />
      </Frame>
  </Step>
  <Step title="Copy the API Secret">
      Click on `Show` button and copy the API SecretÂ to a safe place.
  
      <Frame>
          <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Copy your API Secret" src="/images/notion/image9.png" />
      </Frame>
  </Step>
  <Step title="Navigate to Access Page">
      Navigate to the `Access` menu and click on `Select pages` button.
  
      <Frame>
          <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Navigate to Select Pages" src="/images/notion/image10.png" />
      </Frame>
  </Step>
  <Step title="Select pages to access">
      Select all the pages, which you want to access via this API Secret and click on `Update access` button.
  
      <Frame>
          <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Select pages" src="/images/notion/image11.png" />
      </Frame>
  </Step>
</Steps>

## Find your Version

Your Version can be found at https://developers.notion.com/reference/versioning

## Linking you Account

Upon reaching the Link Account page, select Notion integration from hub and enter the credentials from the previous steps:
  - **API Key**
  - **Version**

Proceed by clicking the Connect button.

    <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Enter Credentials" src="/images/notion/image12.png" />
</Frame>
