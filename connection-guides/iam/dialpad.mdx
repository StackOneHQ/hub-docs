</Warning>

If you've been directed to StackOne to integrate with Dialpad, the following steps will help you understand the process and any necessary actions to configure successful integration.

This guidance assumes that you are an Company Admin within Dialpad.

## Generating an API Key

<Steps>
  <Step title="Login in to Dialpad">
    After logging in to [Dialpad account](https://dialpad.com/login), click [Settings](https://dialpad.com/companysettings) in the left navigation panel.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Admin Settings" src="/images/dialpad/image1.png" />
    </Frame>
  </Step>
  
  <Step title="Select My company">
    Log in to your [Dialpad account](https://dialpad.com/login).Open [Admin Settings](https://dialpad.com/companysettings) and select My company.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Admin Settings" src="/images/dialpad/image2.png" />
    </Frame>
  </Step>

    <Step title="Navigate to API Keys">
        Click Authentication > API Keys in the side menu.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Admin Settings" src="/images/dialpad/image3.png" />
    </Frame>
  </Step>

   <Step title="Add API Key">
        Click Add Key button, Name your key and set the expiration terms and click on Save button.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Admin Settings" src="/images/dialpad/image4.png" />
    </Frame>
  </Step>

     <Step title="Copy the API Key">
        Your new API Key is now created. Copy the API key and store it safely.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Admin Settings" src="/images/dialpad/image5.png" />
    </Frame>
  </Step>

  </Steps>

## Connecting with StackOne

<Steps> 
  <Step title="Enter API Key">
    Upon reaching the Link Account page, enter your API from the previous step.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Admin Settings" src="/images/dialpad/image6.png" />
    </Frame>
  </Step>

  <Step title="Connect">
    Click the Connect button to proceed.
  </Step>
</Steps>
