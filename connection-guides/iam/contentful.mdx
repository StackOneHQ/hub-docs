  Ensure you have Admin privileges within your Contentful organization before proceeding.
</Warning>

If you've been directed to StackOne to integrate with Contentful, follow these steps to configure a successful integration.

Each connection will be associated with a single Space within your Contentful organization.

## Get your Contentful Organization ID

<Steps>
  <Step title="Log in to Contentful">
    Log in to your [Contentful application](https://app.contentful.com)
  </Step>

  <Step title="Expand Side Panel">
    Click the navigation icon in the top-left corner to expand the side panel.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Side Panel" src="/images/contentful/image1.png" />
    </Frame>
  </Step>

  <Step title="Access Organization Settings">
    At the bottom of the side panel, click "Organization settings & subscriptions".
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Organization Settings" src="/images/contentful/image2.png" />
    </Frame>
  </Step>

  <Step title="Copy Organization ID">
    The Organization ID is available in two locations:
    - In the URL: `https://app.contentful.com/account/organizations/{Organization ID}/etc_path`
    - By clicking "Organization information" in the top navigation
    
    Copy and store this Organization ID securely.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Organization Information" src="/images/contentful/image3.png" />
    </Frame>
  </Step>
</Steps>

## Get your Contentful Space ID

<Steps>
  <Step title="Navigate to Spaces">
    Within Organization Settings, select "Spaces" from the top navigation.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Spaces" src="/images/contentful/image4.png" />
    </Frame>
  </Step>

  <Step title="Select Space">
    Scroll through the list of Spaces in your Organization and click the Space you want to associate with this connection.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Select Space" src="/images/contentful/image5.png" />
    </Frame>
  </Step>

  <Step title="Copy Space ID">
    Locate the Space ID in the URL: `https://app.contentful.com/spaces/{Space ID}/home`
    
    Copy and store this Space ID securely.
  </Step>
</Steps>

## Get your Contentful CMA API Token

To access Contentful's Content Management API (CMA), you'll need to generate a CMA Token.

<Steps>
  <Step title="Access Account Settings">
    In your [Contentful application](https://app.contentful.com), click your profile icon in the top-right corner, then select "Account Settings".
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Account Settings" src="/images/contentful/image6.png" />
    </Frame>
  </Step>

  <Step title="Navigate to CMA Tokens">
    Select "CMA Tokens" from the top navigation menu.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="CMA Tokens" src="/images/contentful/image7.png" />
    </Frame>
  </Step>

  <Step title="Create Token">
    Click the "Create personal access token" button.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Create Token" src="/images/contentful/image8.png" />
    </Frame>
  </Step>

  <Step title="Generate Token">
    In the dialog window:
    1. Enter a descriptive name for your token
    2. Set an appropriate expiration date
    3. Click "Generate"
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Generate Token" src="/images/contentful/image9.png" />
    </Frame>
  </Step>

  <Step title="Copy API Token">
    Copy the generated API Token immediately and store it securely. This token will only be displayed once.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Copy API Token" src="/images/contentful/image10.png" />
    </Frame>
  </Step>
</Steps>

## Connecting to StackOne

<Steps>
  <Step title="Enter Credentials">
    On the Link Account page, enter the following credentials that you collected in the previous steps:
    <ul>
      <li>API Token</li>
      <li>Organization ID</li>
      <li>Space ID</li>
    </ul>
    Click the "Connect" button to proceed.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Link Account" src="/images/contentful/image11.png" />
    </Frame>
  </Step>
</Steps>
