If you've been directed to StackOne to integrate with Fivetran, the following steps will help you understand the process and any necessary actions to configure successful integration.

This guidance assumes that you are an Account Administrator in Fivetran.

## Get Your Fivetran API Credentials

<Steps>
  <Step title="Log in to Fivetran">
    Log in to your [Fivetran account](https://fivetran.com)
  </Step>

  <Step title="Navigate to API Key">
    Click your Profile name in the bottom left, then click API Key.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="API Key Navigation" src="/images/fivetran/image1.png" />
    </Frame>
  </Step>

  <Step title="Generate API Key">
    Under API Key, click the Generate new API key button.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Generate API Key" src="/images/fivetran/image2.png" />
    </Frame>
  </Step>

  <Step title="Copy API Key and Secret">
    Copy your displayed API Key and API Secret and store them safely.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="API Key and Secret" src="/images/fivetran/image3.png" />
    </Frame>
  </Step>
</Steps>

## Connecting with StackOne

<Steps>
  <Step title="Enter API Credentials">
    Upon reaching the Link Account page, enter your API Key and API Secret from the previous steps.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Link Account" src="/images/fivetran/image4.png" />
    </Frame>
  </Step>

  <Step title="Connect">
    Proceed by clicking the Connect button.
  </Step>
</Steps>
