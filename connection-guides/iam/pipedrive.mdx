If you've been directed to StackOne to integrate with Pipedrive, the following steps will help you understand the process and any necessary actions to configure successful integration.

## Generating API token and Company domain

<Steps>
  <Step title="Access Company settings">
    To access the Company settings, click on the Profile icon located at the top-right corner of the screen, and select it from the menu that appears.

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Profile icon" src="/images/pipedrive/image1.png" />
    </Frame>
  </Step>

  <Step title="Provide company details">
    Provide your company's name and domain, then click the `Save` button.

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Company details" src="/images/pipedrive/image2.png" />
    </Frame>
  </Step>

  <Step title="Generate API token">
    Go to the Personal Preferences page and navigate to the API tab in order to generate a new API token.

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="API tab" src="/images/pipedrive/image3.png" />
    </Frame>
  </Step>
</Steps>

Congratulations! You are now ready to connect with StackOne.

## Utilizing the API token and Company domain

<Steps>
  <Step title="Enter the Details">
    Upon reaching the Link Account page, enter the gathered details and proceed by clicking the `Connect` button.

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Link Account" src="/images/pipedrive/image4.png" />
    </Frame>
  </Step>
</Steps>
