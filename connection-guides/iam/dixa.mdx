</Warning>

If you've been directed to StackOne to integrate with Dixa, the following steps will help you understand the process and any necessary actions to configure successful integration.

This guidance assumes that you are an Administrator within Dixa.

## Get your Dixa API Key

<Steps>
  <Step title="Login to Dixa">
    After logging in to your [Dixa account](https://accounts.dixa.com), click the settings button on the left, then click Manage > Integrations.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Dixa Settings" src="/images/dixa/image1.png" />
    </Frame>
  </Step>

  <Step title="Navigate to API tokens">
    Under Integrations, click API tokens.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="API Tokens" src="/images/dixa/image2.png" />
    </Frame>
  </Step>

  <Step title="Add API token">
    Click the Add API token button.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Add API Token" src="/images/dixa/image3.png" />
    </Frame>
  </Step>

  <Step title="Enter Token Name">
    Enter a Name for the new token, then click the Save changes button.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Enter Token Name" src="/images/dixa/image4.png" />
    </Frame>
  </Step>

  <Step title="Copy API Key">
    Your new API Key will be displayed. Copy and store it safely.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Copy API Key" src="/images/dixa/image5.png" />
    </Frame>
  </Step>
</Steps>

## Connecting to StackOne

<Steps>
  <Step title="Enter API Key">
    Upon reaching the Link Account page, enter the API Key from the previous step.
  </Step>

  <Step title="Connect">
    Proceed by clicking the `Connect` button.
  </Step>
</Steps>
