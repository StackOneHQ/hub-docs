<Warning>
  Ensure you have Admin privileges for your Leapsome account.
</Warning>

If you've been directed to StackOne to integrate with Leapsome, the following steps will help you understand the process and any necessary actions to configure successful integration.

## Get your Leapsome API Secret

<Steps>
  <Step title="Log in to Leapsome">
    After logging in to [Leapsome](https://www.leapsome.com), click Settings in the left navigation panel.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Settings" src="/images/leapsome/image1.png" />
    </Frame>
  </Step>

  <Step title="Navigate to Integrations">
    Click Settings > Integrations in the left navigation panel.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Integrations" src="/images/leapsome/image2.png" />
    </Frame>
  </Step>

  <Step title="Access API">
    Click API Access in the page that appears.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="API Access" src="/images/leapsome/image3.png" />
    </Frame>
  </Step>

  <Step title="Copy API Secret">
    Copy the displayed API Secret and store it safely. If one does not yet exist, first click the Create API Secret button.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="API Secret" src="/images/leapsome/image4.png" />
    </Frame>
  </Step>
</Steps>

## Get your Leapsome SCIM Token

<Steps>
  <Step title="Navigate to HRIS Integrations">
    Within the Settings > Integrations page, click HRIS Integrations.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="HRIS Integrations" src="/images/leapsome/image5.png" />
    </Frame>
  </Step>

  <Step title="Select SCIM Integration">
    From the displayed tiles, click the tile named SCIM (Azure AD, Okta, ... ).
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="SCIM Integration" src="/images/leapsome/image6.png" />
    </Frame>
  </Step>

  <Step title="Copy SCIM Authentication Token">
    In the window that appears, copy the displayed SCIM Authentication Token and store it safely. If one does not yet exist, click the Update & Synchronize button.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="SCIM Authentication Token" src="/images/leapsome/image7.png" />
    </Frame>
  </Step>
</Steps>

## Connecting to StackOne

<Steps>
  <Step title="Enter API Secret and SCIM Token">
    Upon reaching the Link Account page, enter the API Secret and SCIM Token from the previous steps.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Link Account" src="/images/leapsome/image8.png" />
    </Frame>
  </Step>

  <Step title="Connect">
    Proceed by clicking the Connect button.
  </Step>
</Steps>
