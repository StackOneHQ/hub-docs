<Warning>
  Ensure that your GitHub account has Owner privileges in the organization you are connecting with.
</Warning>

If you've been directed to StackOne to integrate with GitHub, the following steps will help you understand the process and configure a successful integration.

## Log in to GitHub

<Steps>
    <Step title="Log in">
        Log in to your GitHub account at [https://github.com/login](https://github.com/login).

        <Frame>
            <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="GitHub Login Page" src="/images/github/image2.png" />
        </Frame>
    </Step>
</Steps>

## Create a New Personal Access Token

<Steps>
    <Step title="Navigate to Your Profile">
        Click the **Settings** tab.

        <Frame>
            <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="GitHub Settings Navigation" src="/images/github/image3.png" />
        </Frame>
    </Step>

    <Step title="Access Developer Settings">
        Click **Developer settings**.
        <Frame>
            <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Developer Settings Navigation" src="/images/github/image4.png" />
        </Frame>
    </Step>

    <Step title="Access Personal Access Tokens">
        Select **Tokens (classic)** from the **Personal access tokens** dropdown.

        <Frame>
            <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Personal Access Tokens Menu" src="/images/github/image5.png" />
        </Frame>
    </Step>
    
    <Step title="Generate New Token">
        Click the **Generate new token** dropdown.
        Select **Generate new token (classic)** from the dropdown menu.

        <Frame>
            <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Generate New Token Option" src="/images/github/image6.png" />
        </Frame>
    </Step>

    <Step title="Set Token Expiration">
        Select **No expiration** from the Expiration dropdown. 
        <Frame>
            <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Token Expiration Settings" src="/images/github/image7.png" />
        </Frame>
    </Step>

    <Step title="Select Required Scopes">
        Select the following scope required for this IAM integration:
        <ul>
            <li>admin:org</li>
        </ul>
        <Frame>
            <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Token Scope Selection" src="/images/github/image8.png" />
        </Frame>
    </Step>
    
    <Step title="Generate Token">
        Scroll to the bottom of the page and click **Generate token**.
        <Frame>
            <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Generate Token Button" src="/images/github/image9.png" />
        </Frame>
    </Step>

    <Step title="Save Your Personal Access Token">
        Your new **Personal Access Token** will be displayed only once. Make sure to copy and store this value securely.
        <Frame>
            <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Personal Access Token Display" src="/images/github/image10.png" />
        </Frame>
    </Step>
</Steps>

## Find Your Organization 

<Steps>
    <Step title="Access Organizations">
        Select the **Your organizations** tab.
        <Frame>
            <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Organizations Tab" src="/images/github/image11.png" />
       </Frame>
    </Step>
     <Step title="Select Organization">
       You will see a list of organizations here. Copy the **Name** of the organization you wish to connect.
        <Frame>
            <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Organization Selection" src="/images/github/image13.png" />
       </Frame>
    </Step>
</Steps>

## Connecting with StackOne

<Steps>
  <Step title="Enter Credentials">
    On the Link Account page, enter the following credentials from the previous steps:
    - **Personal Access Token**
    - **Organization**

    Click the Connect button to proceed.

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="StackOne Credentials Entry" src="/images/github/image12.png" />
    </Frame>
  </Step>
</Steps>
