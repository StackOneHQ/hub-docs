<Warning>
  Ensure that your Okta account has **API Access Administrator**, **Organization Administrator**, or **Super Administrator** privileges.
</Warning>

If you've been directed to StackOne to integrate with Okta, follow these steps to configure a successful integration.

## Log in to Okta

Log in to your Okta account at https://login.okta.com/

<Frame>
    <img className='rounded-md' style={{ width: '70%', margin: '0 auto', border: '1px solid #efefef' }} alt="Log in to Okta" src="/images/okta/image2.png" />
</Frame>

## Find your Okta Organization URL

Your Organization URL is visible in the browser address bar when logged in to Okta.

For example, if your URL is `https://MY-ORG.okta.com/app/UserHome`, your **Organization URL** is `MY-ORG`.

## Create your Okta API Token

<Steps>
  <Step title="Navigate to your Okta Admin">
    From your Okta homepage, click the **Admin** button in the top-right corner.
  
    <Frame>
        <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Navigate to your Okta Admin" src="/images/okta/image3.png" />
    </Frame>
  </Step>

  <Step title="Navigate to API Tokens">
      In the left navigation menu, select **Security > API**.
  
      <Frame>
          <img className='rounded-md' style={{ width: '50%', margin: '0 auto', border: '1px solid #efefef' }} alt="Navigate to API" src="/images/okta/image4.png" />
      </Frame>

      Then under *API*, select **Tokens**.

      <Frame>
          <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Click API Tokens" src="/images/okta/image5.png" />
      </Frame>
  </Step>

  <Step title="Create a new API Token">
      Click the **Create Token** button.
  
      <Frame>
          <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Create a new API Token" src="/images/okta/image6.png" />
      </Frame>
  </Step>

  <Step title="Enter API Token details">
      Enter a name for the new API Token. For the dropdown "API calls made with this token must originate from", select **Any IP**.
  
      <Frame>
          <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Enter initial API Token details" src="/images/okta/image7.png" />
      </Frame>

      <Info>
          If your Okta instance requires connections only from specific network zones, contact integrations@stackone.com for assistance.
      </Info>
  </Step>

  <Step title="Copy your API Token">
      Copy your **API Token** and store it securely for later use.
  
      <Frame>
          <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Copy your API Token" src="/images/okta/image8.png" />
      </Frame>
  </Step>
</Steps>

## Connecting with StackOne

On the Link Account page, enter your credentials from the previous steps:
  - **Organization URL**
  - **API Token**

Click the Connect button to proceed.

<Frame>
    <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Enter Credentials" src="/images/okta/image9.png" />
</Frame>
