  Ensure that your Articulate - Reach 360 account has Administrator privileges.
</Warning>

If you've been directed to StackOne to integrate with Articulate - Reach 360, the following steps will help you understand the process and any necessary actions to configure a successful integration.

## Log in to Articulate - Reach 360

<Steps>
    <Step title="Log in">
        Log in to your Articulate 360 account at https://360.articulate.com/

        <Frame>
            <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Log in" src="/images/articulate/image1.png" />
        </Frame>
    </Step>

    <Step title="Navigate to Reach 360">
        Click **Reach 360** in the top navigation bar.

        <Frame>
            <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Navigate to Reach 360" src="/images/articulate/image2.png" />
        </Frame>
    </Step>
</Steps>



## Create a New API Key

<Steps>
    
    <Step title="Navigate to API Keys">
        Under *Manage*, click **Settings**.

        <Frame>
            <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Navigate to API Keys1" src="/images/articulate/image3.png" />
        </Frame>

        Under *Settings*, scroll down to *API keys* and click the **Manage API keys** button.
        
        <Frame>
            <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Navigate to API Keys2" src="/images/articulate/image4.png" />
        </Frame>

    </Step>

    
    <Step title="Create New API Key">
        Click the **Generate New Key** button.

        <Frame>
            <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Create New API Key" src="/images/articulate/image5.png" />
        </Frame>
    </Step>

    
    <Step title="Enter API Key Details">
        Enter a **Name** for this new API Key, then click the **Generate Key** button.

        <Frame>
            <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Enter API Key Details" src="/images/articulate/image6.png" />
        </Frame>
    </Step>

    
    <Step title="Copy your API Key">
        Your new **API Key** will be displayed under *Keys Generated*.

        Copy this and store it securely to be used in the next step.

        <Frame>
            <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Copy your API Key" src="/images/articulate/image7.png" />
        </Frame>
    </Step>

</Steps>


## Connecting with StackOne

<Step title="Enter Credentials">
Upon reaching the Link Account page, enter your **API Key** from the previous step.

Proceed by clicking the Connect button.
</Step>
