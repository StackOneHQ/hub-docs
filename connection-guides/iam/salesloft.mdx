If you've been directed to StackOne to integrate with Salesloft, the following steps will help you understand the process and any necessary actions to configure successful integration.

This guidance assumes that you are an Admin within Salesloft.

## Get your Salesloft API Key

<Steps>
  <Step title="Log in to Salesloft">
    Log in to your [Salesloft account](https://accounts.salesloft.com) and click Your Applications.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Your Applications" src="/images/salesloft/image1.png" />
    </Frame>
  </Step>

  <Step title="Create New API Key">
    Within Your Applications, click the Create New button to the right of API Keys.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Create New API Key" src="/images/salesloft/image2.png" />
    </Frame>
  </Step>

  <Step title="Enter API Key Details">
    Under New API Key, enter a Name for this new App and click the Submit button.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Enter API Key Details" src="/images/salesloft/image3.png" />
    </Frame>
  </Step>

  <Step title="Copy API Key">
    Copy your new API Key and store it safely.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Copy API Key" src="/images/salesloft/image4.png" />
    </Frame>
  </Step>
</Steps>

## Connecting to StackOne

<Steps>
  <Step title="Enter API Key">
    Upon reaching the Link Account page, enter the API Key from the previous step.
  </Step>

  <Step title="Connect">
    Proceed by clicking the Connect button.
  </Step>
</Steps>


## Linking your Account

Enter the required fields from the previous steps in your connection form.
