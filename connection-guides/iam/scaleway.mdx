<Warning>
  Ensure your Scaleway Role has the "Access to Organization features" Rule assigned. You can view Scaleway Policies and Rules for your organization on the [Identity and Access Management (IAM) > Policies](https://console.scaleway.com/iam/policies) page.
</Warning>

If you've been directed to StackOne to integrate with Scaleway, follow these steps to configure a successful integration.

## Get Your Scaleway Organization ID

<Steps>
  <Step title="Log in to Scaleway">
    Log in to your [Scaleway Console](https://console.scaleway.com)
  </Step>

  <Step title="Access Organization Settings">
    In your Organization Dashboard, click "Settings" under your Organization name.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Organization Dashboard" src="/images/scaleway/image1.png" />
    </Frame>
  </Step>

  <Step title="Copy Organization ID">
    Click the copy button next to your Organization ID and store it securely.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Copy Organization ID" src="/images/scaleway/image2.png" />
    </Frame>
  </Step>
</Steps>

## Create a Policy for the Scaleway Application

<Steps>
  <Step title="Access IAM">
    In your Scaleway Console, click your Organization icon in the top right corner, then select [IAM](https://console.scaleway.com).
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="IAM" src="/images/scaleway/image3.png" />
    </Frame>
  </Step>

  <Step title="View Policies">
    Click "Policies" in the navigation menu.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Create Policy" src="/images/scaleway/image4.png" />
    </Frame>
  </Step>

  <Step title="Create New Policy">
    Click the "+ Create policy" button.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Create Policy" src="/images/scaleway/image5.png" />
    </Frame>
  </Step>

  <Step title="Enter Policy Details">
    In the "Create a Policy" form, enter the following required information:
    <ul>
      <li>Policy name</li>
      <li>Principal type: Application</li>
      <li>Application: Select or type your application name</li>
    </ul>
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Policy Details" src="/images/scaleway/image6.png" />
    </Frame>
  </Step>

  <Step title="Add Rules">
    Click "Add rules", select "Access to Organization features", then click "Validate".
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Add Rules" src="/images/scaleway/image7.png" />
    </Frame>
  </Step>

  <Step title="Create Policy">
    After configuring your Rule, click "Create policy". Your new Policy will appear in the policies list.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Create Policy Confirmation" src="/images/scaleway/image8.png" />
    </Frame>
  </Step>
</Steps>

## Create a New Scaleway Application

<Steps>
  <Step title="Access Applications">
    In the Identity and Access Management (IAM) section, navigate to [Applications](https://console.scaleway.com).
  </Step>

  <Step title="Create Application">
    Click the "+ Create application" button.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Create Application" src="/images/scaleway/image9.png" />
    </Frame>
  </Step>

  <Step title="Enter Application Details">
    In the "Create an Application" form, provide the following required information:
    <ul>
      <li>Name</li>
      <li>Policy: Select the Policy you created in the previous steps</li>
    </ul>
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Application Details" src="/images/scaleway/image10.png" />
    </Frame>
  </Step>

  <Step title="Create Application">
    Click "Create application" to finalize the process.
  </Step>
</Steps>

## Get Your Scaleway Secret Key

<Steps>
  <Step title="Access API Keys">
    In the Identity and Access Management (IAM) section, navigate to [API Keys](https://console.scaleway.com).
  </Step>

  <Step title="Generate API Key">
    Click the "+ Generate API key" button.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Generate API Key" src="/images/scaleway/image11.png" />
    </Frame>
  </Step>

  <Step title="Enter API Key Details">
    In the "Generate an API Key" form, select the Application you created previously. Add any optional details as needed.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="API Key Details" src="/images/scaleway/image12.png" />
    </Frame>
  </Step>

  <Step title="Generate API Key">
    Click "Generate API key". Your Secret Key will be displayed once. Copy the Secret Key using the copy button and store it securely.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Copy Secret Key" src="/images/scaleway/image13.png" />
    </Frame>
  </Step>

  <Step title="Close">
    Click "Close" to complete the process.
  </Step>
</Steps>

## Connecting with StackOne

<Steps>
  <Step title="Enter Credentials">
    On the Link Account page, enter your Organization ID and Secret Key obtained from the previous steps.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Link Account" src="/images/scaleway/image14.png" />
    </Frame>
  </Step>

  <Step title="Connect">
    Click the "Connect" button to establish the connection.
  </Step>
</Steps>
