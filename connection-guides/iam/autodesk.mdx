  Ensure you have Admin privileges for your Autodesk account.
</Warning>

If you've been directed to StackOne to integrate with Autodesk, the following steps will help you understand the process and any necessary actions to configure successful integration.

This guidance assumes that you are a Primary Admin or Secondary Admin within Autodesk.

## Enable Your Autodesk Developer Account

<Steps>
  <Step title="Gather Details">
    Following the instructions from the article [Autodesk Get Access to a BIM 360 Account](https://aps.autodesk.com), gather the following details:
    - Account Name - located in the top-right when logged in to the B360 Admin
      <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Applications" src="/images/autodesk/image1.png" />
      </Frame>
    - Account ID - located in the URL when logged in to the B360 Admin. This will also be used when connecting with StackOne.

      Example: `https://admin.b360.autodesk.com/admin/{{Account ID}}/projects`

    - Account Admin Email - this can be found on the User Management by User page
  </Step>

  <Step title="Request Activation">
    Send an email to bim360appsactivations@autodesk.com to request activation of your Developer Account. Include the Account Name, Account ID, and Account Admin Email from the step above.
    <ul>
      <li>This may take a few business days to complete. Autodesk will respond via email once complete.</li>
    </ul>
  </Step>
</Steps>

## Create Your Autodesk Application

<Steps>
  <Step title="Access Applications">
    In your Autodesk Platform, click your profile icon in the top-right, then click [Applications](https://aps.autodesk.com).
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Applications" src="/images/autodesk/image2.png" />
    </Frame>
  </Step>

  <Step title="Create Application">
    Click the + Create Application button.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Create Application" src="/images/autodesk/image3.png" />
    </Frame>
  </Step>

  <Step title="Enter Application Details">
    Under Create Application, enter a Name for the new app, and select Traditional Web App. Click the Create button to continue.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Enter Application Details" src="/images/autodesk/image4.png" />
    </Frame>
  </Step>

  <Step title="Copy Credentials">
    Copy your displayed Client ID and Client Secret and store them safely to use in the following steps.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Copy Credentials" src="/images/autodesk/image5.png" />
    </Frame>
  </Step>

  <Step title="Set Callback URL">
    Under General Settings, enter the following as the Callback URL: `https://api.stackone.com/connect/oauth2/autodesk_iam/callback`
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Set Callback URL" src="/images/autodesk/image13.png" />
    </Frame>
  </Step>

  <Step title="Set API Access">
    Under API Access, select the following two APIs required for this integration:
    <ul>
      <li>Autodesk Construction Cloud API</li>
      <li>BIM 360 API</li>
    </ul>
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Set API Access" src="/images/autodesk/image14.png" />
    </Frame>
  </Step>

  <Step title="Save Your New App">
    Press the Save changes button to complete the creation of your new app.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Save Your New App" src="/images/autodesk/image15.png" />
    </Frame>
  </Step>
</Steps>

## Enable Your Autodesk App API Access

<Steps>
  <Step title="Access Settings">
    In your [B360 Admin](https://admin.b360.autodesk.com), click SETTINGS.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Access Settings" src="/images/autodesk/image6.png" />
    </Frame>
  </Step>

  <Step title="Custom Integrations">
    Click Custom Integrations.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Custom Integrations" src="/images/autodesk/image7.png" />
    </Frame>
  </Step>

  <Step title="Add Custom Integration">
    Create a new integration for the StackOne connection by clicking the Add Custom Integration button.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Add Custom Integration" src="/images/autodesk/image8.png" />
    </Frame>
  </Step>

  <Step title="Select Access">
    Under Select Access, enable BIM 360 Account Administration and Document Management, then click the Next button.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Select Access" src="/images/autodesk/image9.png" />
    </Frame>
  </Step>

  <Step title="Developer Selection">
    Select I'm the developer, then click the Next button.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Developer Selection" src="/images/autodesk/image10.png" />
    </Frame>
  </Step> 

  <Step title="BIM 360 Info">
    Under BIM 360 info you need:
    <ul>
      <li>Check the box next to "I have saved the Account ID information securely"</li>
      <li>Enter your App's Client ID from the previous step as the Forge Client ID</li>
      <li>Enter the Name of your App as the App Name</li>
      <li>Click the Save button to complete</li>
    </ul>
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="BIM 360 Info" src="/images/autodesk/image11.png" />
    </Frame>
  </Step>
</Steps>

## Connecting with StackOne

<Steps>
  <Step title="Enter Credentials">
    Upon reaching the Link Account page, enter your credentials from the previous steps:
    <ul>
      <li>Client ID</li>
      <li>Client Secret</li>
      <li>Account ID</li>
    </ul>
    Proceed by clicking the Connect button.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Link Account" src="/images/autodesk/image12.png" />
    </Frame>
  </Step>
</Steps>
