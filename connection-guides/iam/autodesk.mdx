</Warning>

If you've been directed to StackOne to integrate with Autodesk, the following steps will help you understand the process and necessary actions to configure a successful integration.

This guide assumes that you are a Primary Admin or Secondary Admin within Autodesk.

## Enable Your Autodesk Developer Account

<Steps>
  <Step title="Gather Details">
    Following the instructions from the article [Autodesk Get Access to a BIM 360 Account](https://aps.autodesk.com), gather these essential details:
    - Account Name - Located in the top-right when logged into the BIM 360 Admin
      <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Applications" src="/images/autodesk/image1.png" />
      </Frame>
    - Account ID - Found in the URL when logged into the BIM 360 Admin. You'll need this when connecting with StackOne.

      Example: `https://admin.b360.autodesk.com/admin/{{Account ID}}/projects`

    - Account Admin Email - Available on the User Management by User page
  </Step>

  <Step title="Request Activation">
    Send an email to bim360appsactivations@autodesk.com requesting activation of your Developer Account. Include:
    - Account Name
    - Account ID
    - Account Admin Email
    <Note>
      The activation process may take a few business days. Autodesk will notify you via email once complete.
    </Note>
  </Step>
</Steps>

## Create Your Autodesk Application

<Steps>
  <Step title="Access Applications">
    In your Autodesk Platform, click your profile icon in the top-right corner, then select [Applications](https://aps.autodesk.com).
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Applications" src="/images/autodesk/image2.png" />
    </Frame>
  </Step>

  <Step title="Create Application">
    Click the "+ Create Application" button.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Create Application" src="/images/autodesk/image3.png" />
    </Frame>
  </Step>

  <Step title="Enter Application Details">
    In the Create Application form:
    1. Enter a Name for your new app
    2. Select "Traditional Web App"
    3. Click the "Create" button
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Enter Application Details" src="/images/autodesk/image4.png" />
    </Frame>
  </Step>

  <Step title="Copy Credentials">
    Securely copy and store your Client ID and Client Secret. You'll need these for the following steps.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Copy Credentials" src="/images/autodesk/image5.png" />
    </Frame>
  </Step>

  <Step title="Set Callback URL">
    Under General Settings, enter this Callback URL:
    `https://api.stackone.com/connect/oauth2/autodesk_iam/callback`
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Set Callback URL" src="/images/autodesk/image13.png" />
    </Frame>
  </Step>

  <Step title="Set API Access">
    Under API Access, select these two required APIs:
    - Autodesk Construction Cloud API
    - BIM 360 API
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Set API Access" src="/images/autodesk/image14.png" />
    </Frame>
  </Step>

  <Step title="Save Your New App">
    Click the "Save changes" button to complete your app creation.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Save Your New App" src="/images/autodesk/image15.png" />
    </Frame>
  </Step>
</Steps>

## Enable Your Autodesk App API Access

<Steps>
  <Step title="Access Settings">
    In your [BIM 360 Admin](https://admin.b360.autodesk.com), click "SETTINGS".
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Access Settings" src="/images/autodesk/image6.png" />
    </Frame>
  </Step>

  <Step title="Custom Integrations">
    Select "Custom Integrations".
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Custom Integrations" src="/images/autodesk/image7.png" />
    </Frame>
  </Step>

  <Step title="Add Custom Integration">
    Click the "Add Custom Integration" button to create a new integration for StackOne.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Add Custom Integration" src="/images/autodesk/image8.png" />
    </Frame>
  </Step>

  <Step title="Select Access">
    Under Select Access:
    1. Enable BIM 360 Account Administration
    2. Enable Document Management
    3. Click "Next"
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Select Access" src="/images/autodesk/image9.png" />
    </Frame>
  </Step>

  <Step title="Developer Selection">
    Select "I'm the developer" and click "Next".
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Developer Selection" src="/images/autodesk/image10.png" />
    </Frame>
  </Step> 

  <Step title="BIM 360 Info">
    Complete the BIM 360 information:
    1. Check "I have saved the Account ID information securely"
    2. Enter your App's Client ID as the Forge Client ID
    3. Enter your App Name
    4. Click "Save"
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="BIM 360 Info" src="/images/autodesk/image11.png" />
    </Frame>
  </Step>
</Steps>

## Connecting with StackOne

<Steps>
  <Step title="Enter Credentials">
    On the Link Account page, enter these credentials:
    - Client ID
    - Client Secret
    - Account ID

    Click "Connect" to proceed.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Link Account" src="/images/autodesk/image12.png" />
    </Frame>
  </Step>
</Steps>

- Groups
