<Warning>
  Ensure you have Admin privileges within your Elastic organization.
</Warning>

If you've been directed to StackOne to integrate with Elastic, the following steps will help you understand the process and any necessary actions to configure successful integration.

This guidance assumes that you are an Organization Admin within Elastic.

## Get Your Elastic Cloud Organization ID

<Steps>
  <Step title="Log in to Elastic Cloud">
    Log in to your [Elastic Cloud](https://cloud.elastic.co) instance.
  </Step>

  <Step title="Access Organization">
    Click your profile icon in the top right, then click [Organization](https://cloud.elastic.co).
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Elastic Organization" src="/images/elastic/image1.png" />
    </Frame>
  </Step>

  <Step title="Copy Organization ID">
    Copy your displayed Organization ID and store it safely.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Organization ID" src="/images/elastic/image2.png" />
    </Frame>
  </Step>
</Steps>

## Get Your Elastic API Key

<Steps>
  <Step title="Access API Keys">
    While viewing your [Organization](https://cloud.elastic.co) membership, click API keys.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="API Keys" src="/images/elastic/image3.png" />
    </Frame>
  </Step>

  <Step title="Create API Key">
    Click the Create API Key button on the right.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Create API Key" src="/images/elastic/image4.png" />
    </Frame>
  </Step>

  <Step title="Enter API Key Details">
    Under Create API Key:
    <ul>
      <li>Enter a Name for this key</li>
      <li>Set the desired expiration date for this key</li>
      <li>Assign the Organization owner role</li>
      <li>Click the Create API key button</li>
    </ul>
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="API Key Details" src="/images/elastic/image5.png" />
    </Frame>
  </Step>

  <Step title="Copy API Key">
    Copy or download your displayed API Key and store it safely.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Copy API Key" src="/images/elastic/image6.png" />
    </Frame>
  </Step>
</Steps>

## Connecting with StackOne

<Steps>
  <Step title="Enter Credentials">
    Upon reaching the Link Account page, enter your Organization ID and API Key from the previous steps.
  </Step>

  <Step title="Connect">
    Proceed by clicking the Connect button.
  </Step>
</Steps>
