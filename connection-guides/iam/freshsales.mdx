If you've been directed to StackOne to integrate with Freshsales, the following steps will help you understand the process and any necessary actions to configure successful integration.

This guidance assumes that you are an Organization Admin within Freshsales.

## Get your Freshsales Company Domain

<Steps>
  <Step title="Find Your Company Domain">
    This is the same as the CRM domain name required to log in to your Freshsales account. This is also available in the URL when logged into Freshsales.

    Example: If the URL when logged in is `https://mycompany.myfreshworks.com/`, then your Company Domain is `mycompany`.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Company Domain" src="/images/freshsales/image1.png" />
    </Frame>
  </Step>
</Steps>

## Get your Freshsales API Token

<Steps>
  <Step title="Access API Settings">
    While logged in to your Freshsales account, click your profile icon in the top right, then click Settings.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Settings" src="/images/freshsales/image2.png" />
    </Frame>
  </Step>

  <Step title="Navigate to API Settings">
    From Personal Settings, click API Settings in the top navigation bar.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="API Settings" src="/images/freshsales/image3.png" />
    </Frame>
  </Step>

  <Step title="Copy API Key">
    Under CRM API DETAILS, copy your API Key and store it safely.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="API Key" src="/images/freshsales/image4.png" />
    </Frame>
  </Step>
</Steps>

## Connecting to StackOne

<Steps>
  <Step title="Enter Credentials">
    Upon reaching the Link Account page, enter the API Key and Company Domain from the previous steps.
  </Step>

  <Step title="Connect">
    Proceed by clicking the Connect button.
  </Step>
</Steps>
