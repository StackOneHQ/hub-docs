If you've been directed to StackOne to integrate with Veremark, the following steps will help you understand the process and any necessary actions to configure successful integration.

## Getting the API token

<Steps>
    Veremark will provide you with API token for your account, please contact your Account Manager for more information.<br/>
    Refer to the links for <a href="https://help.veremark.com/how-do-i-obtain-an-api-key-for-testing-and-who-should-i-contact-to-ensure-the-engineering-team-receives-it">testing</a> and <a href="https://help.veremark.com/what-are-the-prerequisites-for-launching-the-veremark-api-in-our-production-environment">production</a> API tokens.
</Steps>

## Connecting with StackOne

<Steps>
  <Step title="Enter the Details">
    Upon reaching the Link Account page, enter the `API Token` and proceed by clicking the `Connect` button.
