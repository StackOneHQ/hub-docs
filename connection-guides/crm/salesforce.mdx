<Warning>The following guidance assumes you have Admin privileges for your PLACEHOLDER account.</Warning>

 
## Generating an API Key

Sign in to your account and click the Admin ⚙️ button on the top bar.

In the left menu bar, click on Integrations to expand the menu and then click on API Keys.
​
<img className='rounded-md' style={{ border: '1px solid #efefef' }} alt="API Keys page" src="/images/placeholder/image1.png" />


Click on the ➕ New button to create a new API Key and name it appropriately.


## Use the API Key in the StackOne Hub
Upon reaching the Link Account page, enter the generated API key and proceed by clicking the Connect button.
