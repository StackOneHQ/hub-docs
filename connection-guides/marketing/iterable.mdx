If you've been directed to StackOne to integrate with Iterable, the following steps will help you understand the process and any necessary actions to configure successful integration.

## Find your Iterable region (US or EU)

<Steps>
  <Step title="Login to Iterable">
    Login to your Airship tenant.

    Check your browser's URL bar. The URL will indicate which region you are on:
    `https://app.iterable.com` - US
    `https://app.eu.iterable.com` - EU
  </Step>
</Steps>

## Create an API key

<Steps>
  <Step title="Select Project">
    Click on your account name/icon and select the project you wish to connect to.

    <Frame>
      <img className="rounded-md" width="420px" style={{ margin: '0 auto', border: '1px solid #efefef' }} src="/images/iterable/iterable-01-project.png" />
    </Frame>
  </Step>

  <Step title="Select API Keys">
    Open the Integrations dropdown and select API Keys.

    <Frame>
      <img className="rounded-md" width="420px" style={{ margin: '0 auto', border: '1px solid #efefef' }} src="/images/iterable/iterable-02-integrations.png" />
    </Frame>
  </Step>

  <Step title="Click New API Key" />

  <Step title="Create API Key">
    Enter a **Name** for the API Key
    Select the **Type** as `Server-side`

    Click **Create API Key**
  </Step>

  <Step title="Copy API Key">
    Copy the API Key, this will be used later.
  </Step>
</Steps>

## Retrieve your Master Secret

<Steps>
  <Step title="Reveal Master Secret">
    Within the Project Details panel, select "Click to reveal" next to Master Secret.

    <Frame>
      <img className="rounded-md" style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Reveal Master Secret" src="/images/airship/image5.png" />
    </Frame>
  </Step>

  <Step title="Copy Master Secret">
    Copy the contents of Master Secret.
  </Step>
</Steps>

## Input the collected inputs into the StackOne hub

<Steps>
  <Step title="Enter the Details">
    On the link account page, enter the details you have collected and click the `Connect` button.

    <Frame>
      <img className="rounded-md" style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Link Account" src="/images/airship/image6.png" />
    </Frame>
  </Step>
</Steps>
