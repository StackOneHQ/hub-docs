</Warning>

If you've been directed to StackOne to integrate with Teamtailor, follow these steps to configure a successful integration.

## Generating API Key and Getting Company ID

<Steps>
  <Step title="Navigate to Settings">
    In your Teamtailor account, click on "Settings."

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Settings" src="/images/teamtailor/image1.png" />
    </Frame>
  </Step>

  <Step title="Find API Keys">
    In the left-hand menu under the "Integrations" header, locate "API Keys."

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="API Keys" src="/images/teamtailor/image2.png" />
    </Frame>
  </Step>

  <Step title="Create a New API Key">
    Click the "New API Key" button in the top-right corner of the page.

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="New API Key" src="/images/teamtailor/image3.png" />
    </Frame>
  </Step>

  <Step title="Set Name and Permissions">
    Enter a descriptive name for your API key. Set the scope to `admin` and configure the permissions. Note that only Read permissions are required.

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Set Permissions" src="/images/teamtailor/image4.png" />
    </Frame>
  </Step>

  <Step title="Copy API Key">
    Copy the generated API key and store it in a secure location.

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Copy API Key" src="/images/teamtailor/image5.png" />
    </Frame>
  </Step>

  <Step title="Get Company ID">
    Note your Company ID, which appears in the URL as shown below:

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Company ID" src="/images/teamtailor/image6.png" />
    </Frame>
  </Step>
</Steps>

## Get User Credentials

You'll need to provide a user email and password. We recommend creating a new account specifically for integration purposes.

<Steps>
  <Step title="Create a New Employee">
    Navigate to the "Employees" tab and click the ➕ icon to create a new employee.

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="New Employee" src="/images/teamtailor/image7.png" />
    </Frame>
  </Step>

  <Step title="Fill Out Required Fields">
    Complete all required fields and set the Role to "Recruitment Admin" to ensure the account has appropriate permissions.

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Set Role" src="/images/teamtailor/image8.png" />
    </Frame>
  </Step>

  <Step title="Add Employee">
    Click "Add Employee" to create the account.

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Add Employee" src="/images/teamtailor/image9.png" />
    </Frame>
  </Step>

  <Step title="Create Account">
    The provided email address will receive an invitation. Click "Create Account" to proceed. Note: You may need to log out of your current account to set up the password.

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Create Account" src="/images/teamtailor/image10.png" />
    </Frame>
  </Step>

  <Step title="Find the Invitation Link">
    If you haven't received an invitation, go to the "Employees" tab and click the pencil ✏️ icon to edit the user's profile. You'll find the invitation link there.

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Invitation Link" src="/images/teamtailor/image11.png" />
    </Frame>
  </Step>

  <Step title="Get Account Email and Password">
    Once complete, you should have both the account email address and password ready for the Hub setup.
  </Step>
</Steps>

⚠️ For accounts using Single Sign-On (SSO), the account creation process differs:

### Accounts with Single Sign-On (SSO)

Complete steps 2.1 to 2.2 to create the employee. Then, exclude the employee from SSO under "Exclude Individual Users from SSO Login":

<Steps>
  <Step title="Exclude from SSO">
    After excluding the user from SSO, ensure a password is set. You may need to edit the user's permissions by accessing the "Employees" tab and clicking the gear ⚙️ icon.

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Exclude from SSO" src="/images/teamtailor/image12.png" />
    </Frame>
  </Step>

  <Step title="Set Role">
    On the settings page (similar to step 2.2), set the role to "Recruitment Admin."

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Set Role" src="/images/teamtailor/image13.png" />
    </Frame>
  </Step>
</Steps>

## Two-Factor Authentication (2FA)

<Steps>
  <Step title="Set Up 2FA on Your User Profile">
    In your Teamtailor account, click on "Your Account."

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Settings" src="/images/teamtailor-mfa/image1.png" />
    </Frame>
  </Step>

  <Step title="Access Security Settings">
    Click on "Security."

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Security" src="/images/teamtailor-mfa/image2.png" />
    </Frame>
  </Step>

  <Step title="Set Up Two-Factor Authentication">
    Click the "Setup Two-Factor Authentication" button.

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="2 Step Authentication" src="/images/teamtailor-mfa/image3.png" />
    </Frame>
  </Step>

  <Step title="Save 2FA Secret">
    Scan the QR code with your authenticator app and copy the key displayed below the QR code. Store this key in a secure location for the next step.

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Authentication Secret" src="/images/teamtailor-mfa/image4.png" />
    </Frame>
  </Step>

  <Step title="Enter Verification Code">
    Enter the 6-digit code from your authenticator app to complete the setup.

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Authenticator Code" src="/images/teamtailor-mfa/image5.png" />
    </Frame>
  </Step>

  <Step title="Confirm 2FA Setup">
    Two-Factor Authentication setup is now complete.

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Authentication Complete" src="/images/teamtailor-mfa/image6.png" />
    </Frame>
  </Step>
</Steps>
