If you've been directed to StackOne to integrate with Recruitee, the following steps will help you understand the process and any necessary actions to configure successful integration.

## Generating API Key and getting Company ID

<Steps>
  <Step title="Navigate to Personal API tokens">
    Click on the Settings wheel ⚙️ and scroll down to the Personal API tokens section. Click New token.

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Personal API Tokens" src="/images/recruitee/image1.png" />
    </Frame>
  </Step>

  <Step title="Assign a name to your token">
    Assign a name to your token and specify the Verification Code sent to your email.

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Assign Token Name" src="/images/recruitee/image2.png" />
    </Frame>
  </Step>

  <Step title="Copy your API token and locate your Company ID">
    Copy your API token by clicking Clipboard and locate your Company ID.

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Copy API Token" src="/images/recruitee/image3.png" />
    </Frame>
  </Step>
</Steps>

## Utilizing the API Key and Company ID

<Steps>
  <Step title="Enter the Details">
    Upon reaching the Link Account page, enter the gathered details and proceed by clicking the `Connect` button.

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Link Account" src="/images/recruitee/image4.png" />
    </Frame>
  </Step>
</Steps>
