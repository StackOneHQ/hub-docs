<Note>
  Administrator access is required in your Greenhouse account to manage API credentials.
</Note>
<Steps>
  <Step title="Access API Credentials">
    Click the **Configure** gear icon ⚙️ and navigate to **Dev Center** → **API credentials**.

    <Frame>
      <img src="/images/greenhouse/image1.png" style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Navigate to API credentials" />
    </Frame>
  </Step>

  <Step title="Create Harvest API Key">
    Click **Create new API key** and configure:
    - **API type**: Harvest
    - **Partner**: Custom
    - **Description**: "YOUR_INTEGRATION_NAME_HERE"

    <Frame>
      <img src="/images/greenhouse/image2.png" style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Create Harvest API Key" />
    </Frame>
  </Step>

  <Step title="Save Harvest API Key">
    Click **Manage permissions**, then copy and securely store your API key. Click **I have stored the API key** to continue.

    <Frame>
      <img src="/images/greenhouse/image3.png" style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Save Harvest API Key" />
    </Frame>
  </Step>

  <Step title="Configure Harvest Permissions">
    In the permissions page, click **Select all** to enable all permissions, then click **Save**.

    <Frame>
      <img src="/images/greenhouse/greenhouse_permissions.gif" style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Configure permissions" />
    </Frame>
  </Step>

  <Step title="Create Job Board API Key">
    Return to the API credentials page and create another key:
    - **API type**: Job Board
    - **Partner**: Custom
    - **Description**: "YOUR_INTEGRATION_NAME_HERE"

    Click **Create**.

    <Frame>
      <img src="/images/greenhouse/image4.png" style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Create Job Board API Key" />
    </Frame>
  </Step>

  <Step title="Save Job Board API Key">
    Copy and securely store your Job Board API key. Click **I have stored the API key**.

    <Frame>
      <img src="/images/greenhouse/image5.png" style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Save Job Board API Key" />
    </Frame>
  </Step>

  <Step title="Get Site Admin User ID">
    Navigate to your **Users** page and select a user with Site Admin permissions. Copy the user ID from the URL.

    <Frame>
      <img src="/images/greenhouse/10577285508763.png" style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="User ID in URL" />
    </Frame>
  </Step>
</Steps>

## Linking your Greenhouse Account

Enter these fields in your connection form:
- **Harvest API Key** (from Step 3)
- **Job Board API Key** (from Step 6)
- **Site Admin User ID** (from Step 7)
