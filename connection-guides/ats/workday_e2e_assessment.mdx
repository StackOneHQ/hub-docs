## Create an Integration System using StackOne.

<Steps>
  <Step title="Create a Webhook">
    Set up a webhook in StackOne
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Add Webhook" src="/images/workday-assessment/Add_Webhook.png" />
    </Frame>

    Please ensure that the status of the webhook in the grid below is set to `enabled`
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Webhook Enabled" src="/images/workday-assessment/Webhook_Enabled.png" />
    </Frame>
  </Step>

  <Step title="Add a webhook event">
    To add or edit a webhook event, click on the `Edit event` button and then click on the `+` button next to `created` under ATS Assessments.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Edit Events" src="/images/workday-assessment/Edit_Events.png" />
    </Frame>
    
    When the pop-up appears, please select the `Invited` event under Workday and then click on the `Confirm` button.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Workday Invited Native Event" src="/images/workday-assessment/Select_Workday_Invited_Native_Event.png" />
    </Frame>

    Please remember to click on the `Save` button.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Webhook Event Save" src="/images/workday-assessment/Webhook_Event_Save.png" />
    </Frame>

    <Check>Once you have completed the steps above, the `StackOne_IS_System` Integration System has been successfully created in your linked Workday account.</Check>
  </Step>

  <Step title="To verify on Workday">
    Log into your Workday account and search for `StackOne_IS_System`
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Search IS" src="/images/workday-assessment/verify_1.png" />
    </Frame>
  </Step>

  <Step title="Updated Endpoint Url">
    You will find the Endpoint URL under integration system attributes on the `StackOne_IS_System` page. This URL is used for StackOne to receive notifications about Assessment Invites.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Endpoint Attribute" src="/images/workday-assessment/verify_2.png" />
    </Frame>
  </Step>
</Steps>

<Steps>
  <Step title="Create an assessment category">
    Navigate to <i>Maintain Recruiting Assessment Categories</i> page, Add an assessment category `Technical Assessment Test`.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Maintain Assessment Categories" src="/images/workday-assessment/Assessment_category_1.png" />
    </Frame>

    Click on `OK` & `Done` Button to save & confirm respectively.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Assessment Category" src="/images/workday-assessment/Assessment_category_2.png" />
    </Frame>
  </Step>
</Steps>


## Create Assessment Test in Workday

<Steps>
  <Step title="Create an assessment test">
    Navigate to <i>Maintain Recruiting Assessment Tests</i> page, Add an assessment test `Technical Assessment`.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Maintain Assessment Tests" src="/images/workday-assessment/Assessment_test_1.png" />
    </Frame>

    Enter the test name, select the assessment category created in the previous step, and choose the integration system generated from StackOne, which should be named 'StackOne_IS_System'.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Assessment Tests" src="/images/workday-assessment/Assessment_test_2.png" />
    </Frame>

    Click on `OK` & `Done` Button to save & confirm respectively.
  </Step>

  <Step title="To Edit/Get Assessment_Test_ID">
    Click on the three dot menu, Go to the `View IDs` to view & click on the `Edit Reference ID` to edit the Reference ID.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Edit/View Assessment Test ID" src="/images/workday-assessment/Assessment_test_3.png" />
    </Frame>
  </Step>

  <Step title="Use of Assessment Test ID">
    You'll need `Assessment_Test_ID` i.e `RECRUITING_ASSESSMENT_TEST-6-111` during notification payload configurations.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Assessment Test ID" src="/images/workday-assessment/Assessment_test_4.png" />
    </Frame>
  </Step>
</Steps>


## Create Assessment Status in Workday

<Steps>
  <Step title="Create an assessment status">

## Update Assess Candidate Business Process in Workday

<Steps>
  <Step title="Select to update the Integration System">
    Navigate to the <i>Assess Candidate Business Process</i> page. Click on the `Configure StackOne_IS_System` button.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="BP Config IS" src="/images/workday-assessment/BP_Config_IS.png" />
    </Frame>

    Select `StackOne_IS_System` as an integration and click on the `OK` button.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Select IS on Popup" src="/images/workday-assessment/Select_IS_on_Popup.png" />
    </Frame>
  </Step>

  <Step title="Edit payload attribute values">
    The pop-up will appear, allowing you to select values for the fields that will be included in the payload request when the `Invited` event is triggered from Workday.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Set Values for Fields" src="/images/workday-assessment/Set_Values_for_Fields.png" />
    </Frame>
    
    **Please fill in the following information:**

    - **Candidate First Name:** [First Name]
    - **Candidate Last Name:** [Last Name]
    - **Candidate Email Address:** [Email]
    - **Candidate Reference ID:** [Candidate ID]
    - **Job Application Reference ID:** [Job Requisition as Text]
    - **Assessment Test Reference ID:** [Add Assessment_Test_ID that assessment you want to send to the Candidate]

    Click the `OK` button to save your configurations.
  </Step>
</Steps>

## Move Candiate to the Assessment Stage in Workday

<Steps>
  <Step title="Go to the Job Requisition">
    Navigate to the <i>Job Requisitions</i> page by selecting same Job Requisition from the pop-up where candidate applied for the Job.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="View JR" src="/images/workday-assessment/JR_1.png" />
    </Frame>

    Move to the `Candidate` tab & Click on the `Review` tab where candidate will be visible in below grid.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Candidate Job Apply" src="/images/workday-assessment/JR_5.png" />
    </Frame>
  </Step>

  <Step title="Move Candidate from Review to Assessment stage">
    Mark as checked the Candiate in the grid and Click on the `Move Forward` button.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="View JR" src="/images/workday-assessment/JR_2.png" />
    </Frame>
  </Step>

  <Step title="Select Assessment from the dropdown">
    On the pop-up, you have to select the next stage, which is likely the Assessment Stage. Please select `Assessment` in both the dropdowns.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Candidate Job Apply" src="/images/workday-assessment/JR_3.png" />
    </Frame>

    Click on the `OK` button to move.
  </Step>

  <Step title="Candidate Moved to the Assessment Stage">
    If you navigate to the 'Assessment' tab, the candidate will now appear in the Assessment stage.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="View JR" src="/images/workday-assessment/JR_4.png" />
    </Frame>

    If you are unable to move the candidate directly to the assessment stage, then move the stages one by one.
  </Step>
</Steps>


## Send the Assessment Invite to the Candidate via Start Proxy

<Steps>
  <Step title="Sending via Start Proxy">
    If you are not the Recruiter, then Go to the Recruiter’s account via `Start Proxy`. On the click on the `Awating Action` Link, A pop-up will appear where you‘ll find the Recruiter’s name.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="View Recruiter" src="/images/workday-assessment/NR_1.png" />
    </Frame>
  </Step>

  <Step title="Navigate to the Start Proxy">
    Navigate to the `Start Proxy` page and Select Recruiter from the dropdown.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Start Proxy" src="/images/workday-assessment/NR_2.png" />
    </Frame>

    You'll be directed to the Recruiter's home page where you have to navigate to `My Tasks` page from the top-right corner.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="My Tasks" src="/images/workday-assessment/NR_3.png" />
    </Frame>

    You'll be find the task at the top named with Job Requisition and Candidate name in the sidebar.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Sidebar" src="/images/workday-assessment/NR_4.png" />
    </Frame>

    At the click on the sidebar task, At the right side Candiate's Assessment details will be appear.
  </Step>

  <Step title="Send the Assessment Invitation">
    You need to set the Overall Status as `Invited` and choose the Assessment as `Technical Assessment` from the dropdown menu. Then, click the `Submit` button.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Assessment Form" src="/images/workday-assessment/NR_5.png" />
    </Frame>
  </Step>
</Steps>

## Update Result back to the Workday

<Steps>
  <Step title="Recieved notification payload">
    Use `record_id` as the order `id` in the Update Assessment endpoint with the required body payload:
    ```json
      {
        "score": {
          "value": "906"
        },
        "assessment_date": "2024-08-11T00:00:00.000Z",
        "submission_date": "2024-08-12T00:00:00.000Z",
        "summary": "Assessment is Passed",
        "result": {
          "value": "passed"
        },
        "result_url": "https://www.example.com/content/assessment-result.pdf"
      }
    ```

    Check out the following link for all the details about update assessment result!
    <Card title="Update Assessments Result" icon="link" href="https://docs.stackone.com/reference/ats_update_assessments_result">
      https://docs.stackone.com/reference/ats_update_assessments_result
    </Card>
  </Step>
</Steps>
