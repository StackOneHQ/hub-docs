## Create an integration system using StackOne

<Steps>
  <Step title="Create a Webhook">
    Set up a webhook in StackOne
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Add Webhook" src="/images/workday-assessment/Add_Webhook.png" />
    </Frame>

    Please ensure that the status of the webhook in the grid below is set to `enabled`
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Webhook Enabled" src="/images/workday-assessment/Webhook_Enabled.png" />
    </Frame>
  </Step>

  <Step title="Add a webhook event">
    To add or edit a webhook event, click on the `Edit event` button and then click on the `+` button next to `created` under ATS Assessments.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Edit Events" src="/images/workday-assessment/Edit_Events.png" />
    </Frame>
    
    When the pop-up appears, please select the `Invited` event under Workday and then click on the `Confirm` button.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Workday Invited Native Event" src="/images/workday-assessment/Select_Workday_Invited_Native_Event.png" />
    </Frame>

    Please remember to click on the `Save` button.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Webhook Event Save" src="/images/workday-assessment/Webhook_Event_Save.png" />
    </Frame>

    <Check>Once you have completed the steps above, the `StackOne_IS_System` Integration System has been successfully created in your linked Workday account.</Check>
  </Step>
</Steps>

## Update the result in Workday

<Steps>
  <Step title="Request to update result">
    Use `record_id` as the order `id` in the Update Assessment endpoint with the required body payload:
    ```json
      {
        "score": {
          "value": "906"
        },
        "assessment_date": "2024-08-11T00:00:00.000Z",
        "submission_date": "2024-08-12T00:00:00.000Z",
        "summary": "Assessment is Passed",
        "result": {
          "value": "passed"
        },
        "result_url": "https://www.example.com/content/assessment-result.pdf"
      }
    ```
    Check out the following link for all the details about update assessment result!
    <Card title="Update Assessments Result" icon="link" href="https://docs.stackone.com/reference/ats_update_assessments_result">
      https://docs.stackone.com/reference/ats_update_assessments_result
    </Card>
  </Step>
  <Step title="Updated result in Workday">
    If you received a successful response, you can verify the score in Workday, which should be visible in the task list next to the assessment name.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #EFEFEF' }} alt="Updated Result" src="/images/workday-assessment/Updated_result_1.png" />
    </Frame>
  </Step>
</Steps>
