If you've been directed to StackOne to integrate with JazzHR, the following steps will help you understand the process and any necessary actions to configure successful integration.

## Getting the API key

<Steps>
  <Step title="Open Settings">
    After login, click on Settings which will direct you to the settings page.

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Settings Page" src="/images/jazzhr/image1.png" />
    </Frame>
  </Step>

  <Step title="Navigate to Integrations">
    Once you are directed to Settings, click on the Integrations tab in the subheader.

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Integrations Tab" src="/images/jazzhr/image2.png" />
    </Frame>
  </Step>

  <Step title="Locate API Key">
    Once you are directed to the Integrations page, you’ll find the auto-generated API key in the first section.

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="API Key Section" src="/images/jazzhr/image3.png" />
    </Frame>
  </Step>
</Steps>

## Generating New API key (Optional)

<Steps>
  <Step title="Generate New Key">
    If required, you can generate a new API key by clicking on the GENERATE NEW KEY.

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Generate New Key" src="/images/jazzhr/image4.png" />
    </Frame>
  </Step>

  <Step title="Replace Existing Key">
    After generating the new key, the existing key will be replaced with a new one.

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="New API Key" src="/images/jazzhr/image5.png" />
    </Frame>
  </Step>
</Steps>

NOTE: You can use the auto-generated API key if you don’t want to generate a new one.

## Connecting with StackOne

Upon reaching the Link Account page, enter the gathered details and proceed by clicking the Connect button.

<Frame>
  <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Link Account" src="/images/jazzhr/image6.png" />
</Frame>


## Linking your Account

Enter the required fields from the previous steps in your connection form.
