<Steps>
  <Step title="Access Developer Settings">
    Log in to your Go1 portal and click on your profile tab in the top right corner.

    Select **Integrations** from the dropdown menu, then click **Developers** from the left sidebar.

    <Frame>
      <img src="/images/go1/image1.png" style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Navigate to Integrations" />
    </Frame>
  </Step>

  <Step title="Create New Application">
    Click **Create App** in the top right corner.

    Enter an application name (e.g., "Production Integration").

    Enter the callback URL: `https://api.stackone.com/connect/oauth2/go1/callback`

    Click **Create Application**.

    <Frame>
      <img src="/images/go1/image3.png" style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Create Application" />
    </Frame>
  </Step>

  <Step title="Save Credentials">

## Application Scopes

| Scope | Purpose |
|--------|----------|
| account:read | Read portal information |
| lo:read | Access learning objects |
| portal:read | Read portal configuration |
| user:write | Manage user profiles |
| enrollment:write | Manage user enrollments |

## Linking your Go1 Account

Enter these fields in your connection form:
- **Client ID** (from Step 3)
- **Client Secret** (from Step 3)  
- **Domain** (from Step 4)
- **Application Scopes** (select the scopes required for your integration)

## Available data

This integration has the following [LMS Resources](https://docs.stackone.com/reference/getting-started-with-your-api) available from the provider:
- Users
- Learning Objects
- Enrollments
- Completions
- Portal Configuration
