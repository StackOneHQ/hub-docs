## Configure Application Custom Fields

<Steps>
  <Step title="Search for Manage Templates">
    Type _Manage Templates_ into the search bar and select it from the results
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Search Manage Templates" src="/images/sapsuccessfactors/application-custom-field/sapsf_application_cf_manage_template_search_1.png" />
    </Frame>
  </Step>

  <Step title="Select Job Application template">
    Go to the _Recruiting Management_ tab, select _Job Application_, and choose the default active template to which you want to add a custom fields
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Search Manage Templates" src="/images/sapsuccessfactors/application-custom-field/sapsf_application_cf_mt_Recrut_manage_job_application_option_2.png" />
    </Frame>
  </Step>

  <Step title="Add application custom fields in template">
    Click on `Fields defined`
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Search Manage Templates" src="/images/sapsuccessfactors/application-custom-field/sapsf_appliation_cf_mt_Temaplate_settingS_Field_defined_3.png" />
    </Frame>

    Click the `Add` button and select `Add Custom Field` to add a new custom field block
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Search Manage Templates" src="/images/sapsuccessfactors/application-custom-field/sapsf_aaplication_cf_add_custom_field_button_9.png" />
    </Frame>

    Enter the following field values:

    - **Field ID:** Enter the ID of the field.
    - **Field Type:** Select the type of field (e.g., text).
    - **Field Label:** Enter a label for the field.
    - **Field Description:** Provide a description for the field.
    
    <Info>
      You can select other field's values either as shown in the image or according to your preferences.
    </Info>

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Search Manage Templates" src="/images/sapsuccessfactors/application-custom-field/sapsf_application_cf_add_custom_field_and_enter_details_and_done_4.png" />
    </Frame>
    Click on `Done` to save your changes.

    <Note>
      You can set the position of the custom fields using the top and bottom arrows.
    </Note>
  </Step>

  <Step title="Add application custom fields permissions">
    Click on `Fields Permissions defined`
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Search Manage Templates" src="/images/sapsuccessfactors/application-custom-field/sapsf_application_cf_field_permisssion_5.png" />
    </Frame>

    Enter the following field values:

    - **Field Permission Description:** You can enter any description as this field allows free text.
    - **Field Permission Type:** Select `Write Permission`.
    - **Field Permission Role Name:** Choose the role for which you want to make this custom field visible.
    - **Field Reference ID:** Select the custom field ID that you added in the step above.

    <Info>
      You can choose other field's values based on your preferences.
    </Info>
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Search Manage Templates" src="/images/sapsuccessfactors/application-custom-field/sapsf_application_cf_add_or_edit_permission_and_save_6.png" />
    </Frame>
    Click on `Done` to save your changes.
  </Step>
  
  <Step title="Ensure that you save all of your changes">
    To save your changes, click on `Save & Draft` and then select `Publish`.
    <Warning>
      Changes will only be published if they do not contain any validation errors.
    </Warning>
    
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Search Manage Templates" src="/images/sapsuccessfactors/application-custom-field/sapsf_application_cf_save_as_draft_and_publish_7.png" />
    </Frame>
  </Step>
</Steps>


## Configure Job Requisition Template to Add Custom Fields

<Steps>
  <Step title="Search for Manage Templates">
    Type _Manage Templates_ into the search bar and select it from the results
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Search Manage Templates" src="/images/sapsuccessfactors/application-custom-field/sapsf_application_cf_manage_template_search_1.png" />
    </Frame>
  </Step>

  <Step title="Select Job Requisition template">
    Navigate to _Recruiting Management_ tab > _Job Requisition_,
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Search Manage Templates" src="/images/sapsuccessfactors/application-custom-field/sapsf_application_cf_in_job_req_to_show_8.png" />
    </Frame>

    Select the Job Requisition template to display the application custom fields, such as `Standard Job Requisition`
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Search Manage Templates" src="/images/sapsuccessfactors/application-custom-field/sapsf_application_cf_mt_jr_standard_9.png" />
    </Frame>
  </Step>

  <Step title="Add permissions for application custom fields into the Job Requisition template">
    Click on `General Settings` > `Application Status Configuration` > `Field Permissions defined`
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Search Manage Templates" src="/images/sapsuccessfactors/application-custom-field/sapsf_application_cf_mt_jr_standard_field_permission_10.png" />
    </Frame>

    Click on the `Add` button to add a new field permission block, then Enter the following field values:

    - **Field Permission Description:** You can enter any description as this field allows free text.
    - **Field Permission Type:** Select `Read Permission`.
    - **Field Permission Role Name:** Choose the permission role for which the fields will be displayed.
    - **Field Reference ID:** Select the custom field ID you added in the previous <a href="#configure-application-custom-fields">step 3</a> under "Configure Application Custom Fields" process that needs to be displayed on the UI.
    - **Required Fields:** Choose the fields that you want to make mandatory.
    - **Field Permission Status:** Select the application status indicating when the fields should be displayed.

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Search Manage Templates" src="/images/sapsuccessfactors/application-custom-field/sapsf_application_cf_mt_jr_standard_fp_add_for_custom_11.png" />
    </Frame>
    Click on `Done` to save your changes.
  </Step>

  <Step title="Ensure that you save all of your changes">
    Click on `Save & Draft`, and then click `Publish` to save your changes.
    <Warning>
      Changes will only be published if they do not contain any validation errors.
    </Warning>

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Search Manage Templates" src="/images/sapsuccessfactors/application-custom-field/sapsf_application_cf_save_field_permissions_12.png" />
    </Frame>
  </Step>
</Steps>


## View Application Custom Fields in the Job Application

<Steps>
  <Step title="Go to Job Requisition">
    <Info>
      The custom fields will be displayed for applications associated with job requisition templates for which custom fields are configured.
    </Info>

    Navigate to _Job Requisition_, hover over the specific requisition title to open the Actions menu and then select `View or Edit Requisition`.
    <Accordion title="How to check the Job Requisition Template Name">
      Go to the _Recruiting_ and select any specific job requisition.
      <Frame>
        <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Specific Job Requistion" src="/images/sapsuccessfactors/application-custom-field/sap_sf_bc_recruiting_info_26.png" />
      </Frame>

      Click on the `i` icon to view the Job Requisition Template.
      <Frame>
        <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="JR Info" src="/images/sapsuccessfactors/application-custom-field/sap_sf_bc_recruiting_JR_info_27.png" />
      </Frame>
    </Accordion>

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Search Manage Templates" src="/images/sapsuccessfactors/application-custom-field/sap_sf_bc_recruiting_JR_hove_actions_32.png" />
    </Frame>

    Access the candidate's job application and update their application status as you have configured in the <a href="#configure-job-requisition-template-to-add-custom-fields">step 3</a> in the "Configure Job Requisition Template to Add Custom Fields" process.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Search Manage Templates" src="/images/sapsuccessfactors/application-custom-field/sapsf_applicationcf_goto_job_req_and_goto_stage_13.png" />
    </Frame>

    The newly created custom fields will be visible in the job application
    <Accordion title="If fields are not visible after adding fields & permissions in above steps">
      Navigate to _Admin Center_ and search for `OData API Metadata Refresh and Export`
      <Frame>
        <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="OData API Metadata Refresh and Export" src="/images/sapsuccessfactors/application-custom-field/sap_sf_bc_odata_api_search_38.png" />
      </Frame>

      Click on the `Refresh` button next to Cache.
      <Frame>
        <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Refresh button" src="/images/sapsuccessfactors/application-custom-field/sap_sf_bc_odata_api_cache_refresh_39.png" />
      </Frame>
    </Accordion>

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Search Manage Templates" src="/images/sapsuccessfactors/application-custom-field/sapsf_application_cf_added_cf_on_job_application_visible_14.png" />
    </Frame>

    <Note>
      To adjust the placement of custom fields, refer to <a href="#configure-application-custom-fields">step 3</a> in the "Configure Application Custom Fields" process.
    </Note>
  </Step>
</Steps>
