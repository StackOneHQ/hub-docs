</Info>

## Add a custom field for background check's packages

<Steps>
    <Step title="Add a custom field under user option type">
        Go to the _Custom Fields_ page by selecting _Customise Fields_ from the _Admin_ menu
        <Frame>
            <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Go to Custom Fields page" src="/images/eploy/assess-bcs/eploy_bc_admin_custom_1.png" />
        </Frame>
    </Step>

    <Step title="Go to User Option Types tab">
        Go to the _User Option Types_ tab and click the `New` button to create a new custom option in Eploy.

        <Frame>
            <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Go to user option types tab" src="/images/eploy/assess-bcs/eploy_bc_navigate_to_user_option_type_and_new_2.png" />
        </Frame>

        Enter the Title as `Background Checks` and click on the save button
        <Warning>
            Please use name as `Background Checks` else you won't get the notification from Eploy.
        </Warning>
        <Frame>
            <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="add custom field title" src="/images/eploy/assess-bcs/eploy_bc_user_option_type_4.png" />
        </Frame>
    </Step>
</Steps>
