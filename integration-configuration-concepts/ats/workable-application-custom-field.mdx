## Create Workable Application Custom Field

<Steps>
  <Step title="Click on profile icon">
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Search Create Custom Object" src="/images/workable/custom-field/custom-field-01.png" />
    </Frame>
  </Step>
   <Step title="Click on setting">
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Search Create Custom Object" src="/images/workable/custom-field/custom-field-02.png" />
    </Frame>
  </Step>
  <Step title="Click on workflow">
  scroll down to the bottom of the page and goto the section <b>Candidate custom fields </b> 
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Search Create Custom Object" src="/images/workable/custom-field/custom-field-03.png" />
    </Frame>
  </Step>
   <Step title="Candidate custom fields section">
   Here is the section where you can add custom fields
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Search Create Custom Object" src="/images/workable/custom-field/custom-field-04.png" />
    </Frame>
  </Step>
   <Step title="Fields section">
   Here are different groups of custom fields
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Search Create Custom Object" src="/images/workable/custom-field/custom-field-05.png" />
    </Frame>
  </Step>
   <Step title="Candidate custom fields section">
   Hover over the desired section and click on the <b> + </b> icon to add a new custom field
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Search Create Custom Object" src="/images/workable/custom-field/custom-field-06.png" />
    </Frame>
  </Step>
   <Step title="Custom field form">
   Here is the form to create a custom field
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Search Create Custom Object" src="/images/workable/custom-field/custom-field-07.png" />
    </Frame>
  </Step>
   <Step title="Custom field form">
    Enter a  <b>custom field title</b> choose  a <b>`Field type` </b> and set <b>`Provider = User`</b> from the dropdown menu.
    (The Key thing is the custom field MUST be set to Provider = User)

     click  <b>`Add Custom field`</b> and scroll  to the bottom

    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Search Create Custom Object" src="/images/workable/custom-field/custom-field-08.png" />
    </Frame>
  </Step>
   <Step title="Application Form section">
   click on the <b> `save Changes`</b> button
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Search Create Custom Object" src="/images/workable/custom-field/custom-field-09.png" />
    </Frame>
  </Step>
   <Step title="Created custom field">
   Here is the <b> success </b> message that the custom field is created
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Search Create Custom Object" src="/images/workable/custom-field/custom-field-10.png" />
    </Frame>
  </Step>
   <Step title="Application form preview">
   click on preview and see the custom fields 
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Search Create Custom Object" src="/images/workable/custom-field/custom-field-11.png" />
    </Frame>
  </Step>
   <Step title="Preview candidate custom fields">
   Click on candidate profile  >> scroll down
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Search Create Custom Object" src="/images/workable/custom-field/custom-field-12.png" />
    </Frame>
    See the new created custom field 
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Search Create Custom Object" src="/images/workable/custom-field/custom-field-13.png" />
    </Frame>
  </Step>
   <Step title="Preview candidate custom fields">
   We can also seen <b>Candidate Profile </b>  inside the <b>Candidate custom fields section </b> in the group where we created the custom field
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Search Create Custom Object" src="/images/workable/custom-field/custom-field-14.png" />
    </Frame>
  </Step>
 </Steps>
