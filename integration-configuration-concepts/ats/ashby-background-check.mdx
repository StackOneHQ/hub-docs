## To enable the integration on your Ashby account
<Steps>
  <Step title="Go to Integration">
    Please navigate to Admin in Ashby’s dashboard. Then navigate to Ashby Integrations and click on it
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Maintain Assessment Status" src="/images/ashby-background-check/ashby-bcheck-1.png" />
    </Frame>
  </Step>

  <Step title="To enable the Stackone integration">
    Check the Show unlaunched integrations flag, search for `Stackone` Integration and click on the integration as shown below. A pop up for enabling the `Stackone` integration will appear. Click on Enable `Stackone` integrations.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Edit/View Assessment/Background check Status ID" src="/images/ashby-background-check/ashby-bcheck-2.png" />
    </Frame>

    Once you enable the integration, a pop-up will appear, prompting you to fill in the required fields shown below. 
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Edit/View Assessment/Background check Status ID" src="/images/ashby-background-check/ashby-bcheck-3.png" />
    </Frame>

    Please click on `StackOne API Key`. An input field will appear, allowing you to enter the `External Trigger Token` provided by the StackOne Hub. After entering the `External Trigger Token`, click Update.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Edit/View Assessment/Background check Status ID" src="/images/ashby-background-check/ashby-bcheck-4.png" />
    </Frame>

    Please click on `Ashby API Key`. A search bar will appear, allowing you to enter the name of an existing Ashby API Key. If you want to use an existing one, select it; otherwise, click Create New API Key.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Edit/View Assessment/Background check Status ID" src="/images/ashby-background-check/ashby-bcheck-5.png" />
    </Frame>

    Ashby API Key will be generated and displayed. Please copy your Ashby API Key as this is shown only once
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Edit/View Assessment/Background check Status ID" src="/images/ashby-background-check/ashby-bcheck-6.png" />
    </Frame>
  </Step>
</Steps>


## Adding Background check to Candidate Pipeline.
<Steps>
    <Step title="Navigate to your job dashboard and select a job from the job list">
        <Frame>
        <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Edit/View Assessment/Background check Status ID" src="/images/ashby-background-check/ashby-bcheck-7.png" />
        </Frame>
    </Step>

    <Step title="Select a candidate from candidate pipeline stage">
        <Frame>
        <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Edit/View Assessment/Background check Status ID" src="/images/ashby-background-check/ashby-bcheck-8.png" />
        </Frame>
    </Step>

    <Step title="To select the background check tests">
        Click on `New Activity` to add an background check at this stage. Click on `Assessment` that appears after `New Activity` is clicked.
        <Frame>
        <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Edit/View Assessment/Background check Status ID" src="/images/ashby-background-check/ashby-bcheck-9.png" />
        </Frame>

        You will be redirected to `Add Assessment Activity` pop up. Please click on the `Stackone` assessment in Add Assessment Activity. Select the `Assessment Type` from the dropdown list.
        <Frame>
        <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Edit/View Assessment/Background check Status ID" src="/images/ashby-background-check/ashby-bcheck-10.png" />
        </Frame>

        Once selected, click on the `Add Activity` button at the bottom right corner of the` Add Assessment Activity`.
        <Frame>
        <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Edit/View Assessment/Background check Status ID" src="/images/ashby-background-check/ashby-bcheck-11.png" />
        </Frame>

        You will now see that the background check has been added to the candidate profile with a `Start Assessment` button in the `Activity` section.
        <Frame>
        <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Edit/View Assessment/Background check Status ID" src="/images/ashby-background-check/ashby-bcheck-12.png" />
        </Frame>
    </Step>
</Steps>


## Sending StackOne background check test to a Candidate
<Steps>
    <Step title="To send an background check test to the candidate">
        Click the `Start Assessment` button in the Activity section of the selected candidate. A confirmation pop-up will appear, displaying the selected background check type. You can also modify the background check type using the dropdown list. Click `Start` to send the test.
        <Frame>
        <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Edit/View Assessment/Background check Status ID" src="/images/ashby-background-check/ashby-bcheck-13.png" />
        </Frame>

        The screen displays the background check status along with the date and time it was sent. Since the candidate has not yet completed the assessment, the result will remain pending.
         <Frame>
        <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Edit/View Assessment/Background check Status ID" src="/images/ashby-background-check/ashby-bcheck-14.png" />
        </Frame>
    </Step>
</Steps>
