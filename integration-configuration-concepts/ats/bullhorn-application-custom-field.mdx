## Configure Bullhorn Application Custom Fields

<Steps>
  <Step title="Access the Main Menu">
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Dashboard window" src="/images/bullhorn/custom-field/custom-field-01.png" />
    </Frame>
  </Step>

  <Step title="Open Field Mappings">
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Admin menu" src="/images/bullhorn/custom-field/custom-field-config-01.png" />
    </Frame>
  </Step>

  <Step title="Choose Field Group">
    Scroll down and select the `Job Submission` group to configure the custom field.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Fields Group" src="/images/bullhorn/custom-field/custom-field-config-02.png" />
    </Frame>
  </Step>

  <Step title="Review Job Submission Fields">
    Review the job submission-related fields and select an unused one to configure.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Fields Group" src="/images/bullhorn/custom-field/custom-field-config-03.png" />
    </Frame>
  </Step>

  <Step title="Configure Field Settings">
    In the custom fields configuration form, set your desired values and click the <b>`Save`</b> button to apply your changes.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Fields Group" src="/images/bullhorn/custom-field/custom-field-config-04.png" />
    </Frame>
  </Step>

  <Step title="Save Field Configuration">
    Return to the top of the page and click the <b>`Save`</b> button to finalize your changes.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Fields Group" src="/images/bullhorn/custom-field/custom-field-config-05.png" />
    </Frame>
  </Step>

  <Step title="Confirm Configuration">
    A confirmation message will appear once your configuration has been successfully saved.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Fields Group" src="/images/bullhorn/custom-field/custom-field-config-06.png" />
    </Frame>
  </Step>
</Steps>

## Using Bullhorn Application Custom Fields

<Steps>
  <Step title="Access the Main Menu">
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Dashboard window" src="/images/bullhorn/custom-field/custom-field-01.png" />
    </Frame>
  </Step>

  <Step title="Navigate to Candidates Section">
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Admin menu" src="/images/bullhorn/custom-field/custom-field-02.png" />
    </Frame>
  </Step>

  <Step title="Open Column Settings">
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Candidate table" src="/images/bullhorn/custom-field/custom-field-03.png" />
    </Frame>
  </Step>

  <Step title="Select Custom Fields to Display">
    In the custom fields table, scroll down and select the desired custom fields by checking their corresponding checkboxes in the dropdown menu.  
    Then, click the <b>`Save`</b> button to apply your changes.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Custom fields column" src="/images/bullhorn/custom-field/custom-field-04.png" />
    </Frame>
  </Step>

  <Step title="Arrange Custom Fields in Table">
    The custom field will now appear in the Candidate Table. You can reposition it by dragging and dropping to your preferred location.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Custom fields view in the table" src="/images/bullhorn/custom-field/custom-field-05.png" />
    </Frame>
  </Step>

  <Step title="Access Candidate Details">
    In the candidate table, click on any candidate to view their detailed profile.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Candidate profile" src="/images/bullhorn/custom-field/custom-field-06.png" />
    </Frame>
  </Step>

  <Step title="Enter Edit Mode">
    In the candidate profile view, click the <b>`Edit`</b> button to modify or add custom field values.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Candidate profile with multiple fields" src="/images/bullhorn/custom-field/custom-field-07.png" />
    </Frame>
  </Step>

  <Step title="Locate Custom Field Section">
    Scroll down in the candidate profile edit section to locate and modify the custom field.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Candidate profile edit section" src="/images/bullhorn/custom-field/custom-field-08.png" />
    </Frame>
  </Step>

  <Step title="View and Edit Custom Field">
    The custom field will be displayed in the candidate profile edit section, where you can view and modify its value.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Custom field" src="/images/bullhorn/custom-field/custom-field-09.png" />
    </Frame>
  </Step>
</Steps>
