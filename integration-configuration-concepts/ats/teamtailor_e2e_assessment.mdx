## Create a Partner Integration in Teamtailor

<Steps>
  <Step title="Navigate to Integration's hub">
    Go to the <i>Integration's Hub</i> page by selecting the Settings option in the top-right corner.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Navigate to Integration's hub" src="/images/teamtailor-assessment/create_a_partner_1.png" />
    </Frame>

    To create a new partner, click on the `Create A New Partner` button.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Create A New Partner" src="/images/teamtailor-assessment/create_a_partner_2.png" />
    </Frame>
  </Step>

  <Step title="Create a New Partner">
    Please enter the name of your partner.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Partner Name" src="/images/teamtailor-assessment/create_a_partner_3.png" />
    </Frame>

    Select `Assessment` from the Marketplace category dropdown. Ensure the capability to make assessments is checked, then click the `Save` button.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Category Should be Assessment" src="/images/teamtailor-assessment/create_a_partner_4.png" />
    </Frame>

    No changes are necessary for the Webhook title fields, which default to `Send` for the verb and `webhook` for the noun. These will be used to send the assessment notifications.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="New Partner Created" src="/images/teamtailor-assessment/create_a_partner_5.png" />
    </Frame>

    You will now notice that the new partner has been successfully created.
  </Step>
</Steps>


## Activate Partner Integration

<Steps>
  <Step title="To get Partner Credentials">
    On the Grid, alongside your partner Integrationâ€™s name, you will find the **Integration ID**. By clicking on <a>Show details</a>, you can access the **API Key** and **Webhook Signature**.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Get Partner credentials" src="/images/teamtailor-assessment/activate_int_marketplace_1.png" />
    </Frame>

    Integration ID and API Key are necessary to configure at the StackOne while enabling Integration.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Id, API Key & Signature" src="/images/teamtailor-assessment/activate_int_marketplace_2.png" />
    </Frame>

    By using Webhook Signature you can activate the integration from the Marketplace, allowing recruiters to select and send assessment notifications to candidates.
  </Step>

  <Step title="To activate partner integration from the Marketplace activations">
    Navigate to the <i>Marketplace Activations</i> page located in the sidebar.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Marketplace Activation's page" src="/images/teamtailor-assessment/activate_int_marketplace_3.png" />
    </Frame>

    Your Partner Integration will be displayed on the Grid. Click the `View settings` button to add or edit the `Integration Webhook Signature`.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="View Settings" src="/images/teamtailor-assessment/activate_int_marketplace_4.png" />
    </Frame>

    The `Integration Webhook Signature` will be used to verify requests at the Partner's end for sending notifications.
  </Step>

  <Step title="To Activate Integration from MarketPlace page">
    If your Partner integration status is set to `Published`, it will be visible on the public Marketplace page.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="From Marketplace page" src="/images/teamtailor-assessment/activate_int_marketplace_5.png" />
    </Frame>

    Click on the Integration, then click on `Go to Settings` to activate the integration.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Go to Activation" src="/images/teamtailor-assessment/activate_int_marketplace_6.png" />
    </Frame>
  </Step>
</Steps>


## Send the assessment invitation to the candidate

<Warning>
  The following guidance assumes that you have configured with the StackOne and enabled the Webhook to receive notifications.
</Warning>

<Steps>
  <Step title="Select Candidate from the Job">
    Navigate to the <i>Job's</i> page and move one of the candidate to the specific stage where you want to trigger assessment.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Maintain Assessment Status" src="/images/teamtailor-assessment/send_assess_1.png" />
    </Frame>

    Click on the candidate to open their application. Then, click on the three-dot more options in the lower bar and select `Add Integration`.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Maintain Assessment Status" src="/images/teamtailor-assessment/send_assess_2.png" />
    </Frame>
  </Step>

  <Step title="To Select & Send the Assessment">
    Select the partner integration you want to use to send the assessment to the candidate.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Edit/View Assessment Status ID" src="/images/teamtailor-assessment/send_assess_3.png" />
    </Frame>

    A pop-up will display a text field with the assessment test ID that is configured at StackOne.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Edit/View Assessment Status ID" src="/images/teamtailor-assessment/send_assess_4.png" />
    </Frame>

    Click the `Send webhook` button to dispatch the notification, and you will see a success response on the same page.
    <Frame>
      <img className='rounded-md' style={{ margin: '0 auto', border: '1px solid #efefef' }} alt="Edit/View Assessment Status ID" src="/images/teamtailor-assessment/send_assess_5.png" />
    </Frame>
  </Step>
</Steps>
