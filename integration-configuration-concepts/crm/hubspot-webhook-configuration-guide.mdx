## To Get External Trigger Token
<Steps>
  <Step title="To get the External Trigger Token">
     From the StackOne hub, you will receive the `External Trigger URL` needed to enable the HubSpot webhook.
    <Frame>
     ![Trigger URL](/images/hubspot-webhook-configuration/image1.png)
    </Frame>
  </Step>
</Steps>

## To Enable Webhook Events
<Steps>
  <Step title="Create or Use a HubSpot App">
   - Go to [HubSpot Developer Portal](https://developers.hubspot.com).
   - Select **Apps** from the navigation panel.
   - Click on **Create app** to create a new app or select an existing app.
   <Frame>
     ![Create app](/images/hubspot-webhook-configuration/image3.png)
    </Frame>
  </Step>
  <Step title="Navigate to Webhooks">
     - Click **Webhooks** from the left menu.
     <Frame>
     ![Navigate to Webhooks](/images/hubspot-webhook-configuration/image4.png)
    </Frame>
  </Step>
  <Step title="Set the Webhook URL">
     - Add your `External Trigger URL` in the Target URL field.
     <Frame>
     ![Create app](/images/hubspot-webhook-configuration/image5.png)
    </Frame>
     - Click **Save** to save the changes.
     <Frame>
     ![Create app](/images/hubspot-webhook-configuration/image6.png)
    </Frame>
  </Step>
</Steps>
